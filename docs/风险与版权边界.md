# 风险与版权边界

## 基本原则

SceneForge 的目标是经典影视 / 热点片段的动画电影化再创作，而不是原片复刻或明星脸换皮。

## 参考对象必须先明确

进入流程前必须判断本次参考的是：

1. 原著 / 母题 / 神话 / 历史典故。
2. 具体影视版本。

两者都支持，但不能混着默认处理。

## 改编强度四档

1. 原样保留型：适合公版原著、民间神话、传统故事或明确授权内容。
2. 大部分保留型：适合风险相对低且经典认知强的片段。
3. 部分改写型：默认模式，保留情绪核和认知锚点，主体重新表达。
4. 重构扩写型：适合高风险强 IP、近期热播、明星绑定强的片段。

## 角色外观参考强度

1. 高锚点参考型：适合西游记等大众集体记忆强的公版/母题内容。
2. 中度参考型：适合一般经典影视角色，保留气质、身份、服装轮廓和标志元素。
3. 低参考 / 原创转译型：适合高风险强 IP、近期热点、明星脸辨识度过高的内容。

## 正式表达建议

正式提示词和发布文案中，不建议直接使用“皮克斯官方风格”“Pixar 同款”等表达。

推荐使用：

> 高品质 3D 动画电影质感，圆润但有辨识度的角色比例，夸张但真实的表情系统，温暖细腻的电影灯光，适合全年龄动画电影的原创角色设计。

## 关于内部讨论与正式输出

- 在内部设计讨论中，可以用“皮克斯风格”作为沟通捷径来表达目标气质。
- 在正式 Skill 输出、提示词、发布文案和复盘文本中，统一改写为通用动画电影化表述。
- 对于搞笑、夸张或鬼畜路线，推荐使用：
  - 喜剧动画化
  - 夸张搞笑化
  - 鬼畜离谱化
  - 高品质 3D 动画电影质感的搞笑改编
