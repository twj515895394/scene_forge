# 分类标签

技能以五个规范分类角色的术语说话。此文件将这些角色映射到此仓库问题追踪器中使用的实际标签字符串。

| mattpocock/skills 中的标签 | 我们的追踪器中的标签 | 含义 |
| -------------------------- | -------------------- | ---------------------------------------- |
| `needs-triage`             | `needs-triage`       | 维护者需要评估此问题                     |
| `needs-info`               | `needs-info`         | 等待报告者提供更多信息                   |
| `ready-for-agent`          | `ready-for-agent`    | 完全指定，AFK 代理就绪                   |
| `ready-for-human`          | `ready-for-human`    | 需要人工实现                             |
| `wontfix`                  | `wontfix`            | 不会被处理                               |

当技能提到一个角色（如"应用 AFK 就绪分类标签"），使用此表中对应的标签字符串。

编辑右列以匹配你实际使用的词汇。
