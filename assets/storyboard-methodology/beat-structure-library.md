# Beat Structure Library

asset_type: executable_reference
runtime_readable: true

```yaml
beat_required_fields:
  beat_id:
  beat_purpose:
  duration_range_seconds:
  action_progression:
  emotion_progression:
  information_release:
  space_change:
  continuity_from_previous_beat:
  continuity_to_next_beat:
  suggested_shot_count:
```

说明：

- Beat 是节奏骨架，不是普通剧情摘要。
- Beat 必须能同时回答动作推进、情绪推进和信息释放三件事。
- 若一个 Beat 无法稳定映射到镜头节奏，应先补结构再出分镜。
