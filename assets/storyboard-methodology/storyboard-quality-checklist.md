# Storyboard Quality Checklist

asset_type: executable_reference
runtime_readable: true

```yaml
storyboard_quality_dimensions:
  - 叙事检查
  - 节奏检查
  - 动作检查
  - 情绪检查
  - 空间检查
  - 首尾检查
  - 资产检查
  - 风格检查
  - 视频可执行性检查
```

最小检查问题：

1. 是否每个 unit 都有 opening / closing anchor。
2. 是否动作链与情绪链都能连续交接。
3. 是否存在空间跳轴或角色站位漂移。
4. 是否控制版与风格版共用同一套 shot 数据。
5. 是否存在会把故事板痕迹直接带进视频提示词的风险。
