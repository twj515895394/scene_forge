# `ink_poetic_2d` 负向边界（Negative Constraints）

## 禁止项

- 不要把水墨仅当滤镜覆盖。
- 不要让形式化压垮可读性。
- 不要加入过强现代潮流图形语汇。

## 谨慎项

- 当前仍是实验风格，最大风险是空转和审美化过度。
- 与 `poetic_2d_fantasy` 的边界在于它更克制、更卷轴、更东方写意。

## 当前验证不足点

- 该风格当前仍属于 Exploration Pool，资料深度已补齐到可运行水平，但尚未经过系统性的真实项目验证。
- 使用时应优先选择该风格真正擅长的题材，避免把它当成通用风格包。
- 若后续测试发现输出与相邻风格高度重叠，应优先回到本包边界定义做收紧。
