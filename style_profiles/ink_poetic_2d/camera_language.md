# `ink_poetic_2d` 镜头语言（Camera Language）

## 核心方向

- 慢节奏
- 停顿
- 卷轴式展开
- 场景意象转场

## 镜头偏好

- 慢节奏 应成为稳定可复用的镜头选择倾向。
- 停顿 应成为稳定可复用的镜头选择倾向。
- 卷轴式展开 应成为稳定可复用的镜头选择倾向。
- 场景意象转场 应成为稳定可复用的镜头选择倾向。

## 节奏习惯

- 镜头切换优先服务 含蓄 与 凝神 的情绪推进。
- 当项目进入高潮段时，应优先保留最能代表 `ink_poetic_2d` 的镜头组织，而不是借用通用快切。
- 镜头运动、停顿和转场都要帮助观众读出 水墨抒情动画 的独特节拍。
