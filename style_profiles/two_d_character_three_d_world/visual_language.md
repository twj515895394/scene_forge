# `two_d_character_three_d_world` 视觉语言（Visual Language）

## 核心方向

- 2D 人物轮廓与 3D 空间纵深并置
- 层次反差明显
- 角色图形性强
- 环境电影化

## 可直接继承的视觉锚词

- 2D 人物轮廓与 3D 空间纵深并置
- 层次反差明显
- 角色图形性强
- 环境电影化

## 设计要求

- 角色、场景和核心道具都要服务于“2D角色+3D场景混合”的主情绪与空间秩序。
- 视觉变化应围绕 奇幻冒险、跨维度世界 等题材展开，而不是平均化套模板。
- 同一镜头内保持一个主视觉逻辑，避免混入相邻风格的冲突语言。
- 关键画面必须让人一眼读出该风格的媒介属性和情绪基线。
