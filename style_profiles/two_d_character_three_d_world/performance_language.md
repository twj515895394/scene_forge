# `two_d_character_three_d_world` 表演语言（Performance Language）

## 核心方向

- 角色表演更图形化
- 环境反馈更电影化
- 夸张与真实需要共存
- 接触关系必须可信

## 表演偏好

- 角色表演更图形化 应通过具体反应、姿态或停顿被看见。
- 环境反馈更电影化 应通过具体反应、姿态或停顿被看见。
- 夸张与真实需要共存 应通过具体反应、姿态或停顿被看见。
- 接触关系必须可信 应通过具体反应、姿态或停顿被看见。

## 不适合的表演倾向

- 不要让 2D 角色像贴纸悬浮。
- 不要只追求风格差异而忽略接触可信度。
- 不要让复杂镜头运动毁掉媒介统一性。
