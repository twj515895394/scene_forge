# `two_d_character_three_d_world` 镜头语言（Camera Language）

## 核心方向

- 需要解决镜头运动与 2D 角色一致性
- 空间调度比单镜头帅更重要
- 透视变化要照顾角色图形稳定性
- 前后景关系决定成立度

## 镜头偏好

- 需要解决镜头运动与 2D 角色一致性 应成为稳定可复用的镜头选择倾向。
- 空间调度比单镜头帅更重要 应成为稳定可复用的镜头选择倾向。
- 透视变化要照顾角色图形稳定性 应成为稳定可复用的镜头选择倾向。
- 前后景关系决定成立度 应成为稳定可复用的镜头选择倾向。

## 节奏习惯

- 镜头切换优先服务 角色表演更图形化 与 环境反馈更电影化 的情绪推进。
- 当项目进入高潮段时，应优先保留最能代表 `two_d_character_three_d_world` 的镜头组织，而不是借用通用快切。
- 镜头运动、停顿和转场都要帮助观众读出 2D角色+3D场景混合 的独特节拍。
