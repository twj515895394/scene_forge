# `wild_comedy_2d` 负向边界（Negative Constraints）

## 禁止项

- 不要只有噪音没有笑点结构。
- 不要套入严肃纪实对白节奏。
- 不要为了狂野而失去因果清晰度。

## 谨慎项

- 当前仍是实验风格，验证不足点在于它很容易退化成无节制乱闹。
- 与 `dreamworks_like` 的边界在于它是 2D 崩坏形变喜剧，不是 3D 商业喜剧结构。

## 当前验证不足点

- 该风格当前仍属于 Exploration Pool，资料深度已补齐到可运行水平，但尚未经过系统性的真实项目验证。
- 使用时应优先选择该风格真正擅长的题材，避免把它当成通用风格包。
- 若后续测试发现输出与相邻风格高度重叠，应优先回到本包边界定义做收紧。
