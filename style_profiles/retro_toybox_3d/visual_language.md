# `retro_toybox_3d` 视觉语言（Visual Language）

## 核心方向

- 复古塑料和木质玩具感
- 鲜艳但偏旧时代配色
- 可收藏品质感
- 箱庭空间比例

## 可直接继承的视觉锚词

- 复古塑料和木质玩具感
- 鲜艳但偏旧时代配色
- 可收藏品质感
- 箱庭空间比例

## 设计要求

- 角色、场景和核心道具都要服务于“复古玩具箱 3D”的主情绪与空间秩序。
- 视觉变化应围绕 玩具世界、收藏柜冒险 等题材展开，而不是平均化套模板。
- 同一镜头内保持一个主视觉逻辑，避免混入相邻风格的冲突语言。
- 关键画面必须让人一眼读出该风格的媒介属性和情绪基线。
