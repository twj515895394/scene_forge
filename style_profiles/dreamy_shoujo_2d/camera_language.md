# `dreamy_shoujo_2d` 镜头语言（Camera Language）

## 核心方向

- 柔性特写
- 情绪蒙太奇
- 轻飘过渡
- 主观镜头比客观交代更重要

## 镜头偏好

- 柔性特写 应成为稳定可复用的镜头选择倾向。
- 情绪蒙太奇 应成为稳定可复用的镜头选择倾向。
- 轻飘过渡 应成为稳定可复用的镜头选择倾向。
- 主观镜头比客观交代更重要 应成为稳定可复用的镜头选择倾向。

## 节奏习惯

- 镜头切换优先服务 心动 与 犹豫 的情绪推进。
- 当项目进入高潮段时，应优先保留最能代表 `dreamy_shoujo_2d` 的镜头组织，而不是借用通用快切。
- 镜头运动、停顿和转场都要帮助观众读出 少女梦幻叙事动画 的独特节拍。
