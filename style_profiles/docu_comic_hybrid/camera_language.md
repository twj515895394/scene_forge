# `docu_comic_hybrid` 镜头语言（Camera Language）

## 核心方向

- 纪实段落与批注段落交替
- 现场观察先于观点覆盖
- 批注要跟随画面重心
- 镜头不宜过分炫技

## 镜头偏好

- 纪实段落与批注段落交替 应成为稳定可复用的镜头选择倾向。
- 现场观察先于观点覆盖 应成为稳定可复用的镜头选择倾向。
- 批注要跟随画面重心 应成为稳定可复用的镜头选择倾向。
- 镜头不宜过分炫技 应成为稳定可复用的镜头选择倾向。

## 节奏习惯

- 镜头切换优先服务 观察 与 解释 的情绪推进。
- 当项目进入高潮段时，应优先保留最能代表 `docu_comic_hybrid` 的镜头组织，而不是借用通用快切。
- 镜头运动、停顿和转场都要帮助观众读出 纪录感+漫画批注混合 的独特节拍。
