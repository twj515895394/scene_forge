# `docu_comic_hybrid` 负向边界（Negative Constraints）

## 禁止项

- 不要用纯讲稿替代镜头组织。
- 不要让批注层像 PPT 覆盖一切。
- 不要失去纪实观察的可信度。

## 谨慎项

- 当前仍是实验风格，风险在于观点层过多、镜头层过弱。
- 与 `social_fastcut_motion_comic` 的边界在于它有纪实基底和观察性镜头。

## 当前验证不足点

- 该风格当前仍属于 Exploration Pool，资料深度已补齐到可运行水平，但尚未经过系统性的真实项目验证。
- 使用时应优先选择该风格真正擅长的题材，避免把它当成通用风格包。
- 若后续测试发现输出与相邻风格高度重叠，应优先回到本包边界定义做收紧。
