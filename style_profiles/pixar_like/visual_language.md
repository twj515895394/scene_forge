# `pixar_like` 视觉语言（Visual Language）

## 核心方向

- 圆润但不幼稚的体块组织
- 高可读轮廓与亲和比例
- 动画比例搭配可信材质
- 温暖、细腻、适合全年龄受众的电影画面气质

## 可直接继承的视觉锚词

- 高品质 3D 动画电影质感
- 角色魅力优先的动画角色设计
- 圆润但有辨识度的角色比例
- 高可读表情系统
- 电影级材质与灯光统一
- feature animation character sheet

## 角色设计要求

- 轮廓必须一眼可认
- 头身比服务角色魅力，不走半写实真人
- 眼睛、眉骨、嘴型必须支持细腻情绪变化
- 表情区要能承载微表情，而不只是大开大合的喜剧脸

## 场景与道具要求

- 场景设计化，而不是现实照搬
- 道具角色化，而不是普通商品图
- 主场景、角色、核心道具必须属于同一视觉宇宙
- 色彩和材质统一优先于局部花哨
