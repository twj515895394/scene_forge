# `warm_humanist_cinematic` 视觉语言（Visual Language）

## 核心方向

- 自然暖光
- 生活细节
- 真实空间中的柔软秩序
- 不夸张但有温度的色彩

## 可直接继承的视觉锚词

- 自然暖光
- 生活细节
- 真实空间中的柔软秩序
- 不夸张但有温度的色彩

## 设计要求

- 角色、场景和核心道具都要服务于“温暖治愈生活电影感”的主情绪与空间秩序。
- 视觉变化应围绕 家庭关系、普通人小愿望 等题材展开，而不是平均化套模板。
- 同一镜头内保持一个主视觉逻辑，避免混入相邻风格的冲突语言。
- 关键画面必须让人一眼读出该风格的媒介属性和情绪基线。
