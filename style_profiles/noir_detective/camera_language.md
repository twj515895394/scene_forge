# `noir_detective` 镜头语言（Camera Language）

## 核心方向

- 观察性机位
- 长廊和门框构图
- 低速推进
- 悬疑信息延迟释放

## 镜头偏好

- 观察性机位 应成为稳定可复用的镜头选择倾向。
- 长廊和门框构图 应成为稳定可复用的镜头选择倾向。
- 低速推进 应成为稳定可复用的镜头选择倾向。
- 悬疑信息延迟释放 应成为稳定可复用的镜头选择倾向。

## 节奏习惯

- 镜头切换优先服务 疲惫 与 怀疑 的情绪推进。
- 当项目进入高潮段时，应优先保留最能代表 `noir_detective` 的镜头组织，而不是借用通用快切。
- 镜头运动、停顿和转场都要帮助观众读出 黑色侦探电影感 的独特节拍。
