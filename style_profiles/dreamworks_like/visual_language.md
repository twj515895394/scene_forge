# `dreamworks_like` 视觉语言（Visual Language）

## 核心方向

- 轮廓更利落，态度更强
- 角色设计允许更锋利的个性差异
- 色彩更跳、更直接服务娱乐感
- 整体是强商业喜剧 3D，而不是温吞抒情路线

## 可直接继承的视觉锚词

- 高冲突感 3D 动画电影质感
- 角色态度鲜明的动画角色设计
- 利落剪影与更强姿态线
- 更大开大合的表情系统
- 商业娱乐感更强的色彩与材质组织
- comedic feature animation character sheet

## 角色设计要求

- 轮廓一眼能看出角色立场和气质
- 姿态线优先于温和圆润感
- 表情区要支持夸张反应、嘴角戏和挑眉式冲突
- 同场角色之间应有明显体型、脸型或姿态反差

## 场景与道具要求

- 场景要服务冲突和包袱落点，不要太均匀平顺
- 道具可承担笑点、反差或节奏触发功能
- 颜色与材质要帮助区分阵营、态度和能量级别
- 主场景、角色、核心道具要统一在“热闹但可控”的视觉宇宙中
