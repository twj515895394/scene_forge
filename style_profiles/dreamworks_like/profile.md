# `dreamworks_like` 风格总览（Profile）

```yaml
style_profile:
  director_style_id: dreamworks_like
  director_style_version: v1
  style_family: 3d_animation
  summary: >
    以外放喜剧冲突、强角色态度、快速反应、鲜明剪影和更明显的商业娱乐节奏为核心的
    3D 动画电影化风格。
  suitable_topics:
    - 嘴仗喜剧
    - 群像冲突短片
    - 反差搭档桥段
    - 强对抗动作喜剧
  recommended_performance_styles:
    - cinematic_comedy
    - exaggerated_comedy
  required_profile_files:
    - style_profiles/dreamworks_like/visual_language.md
    - style_profiles/dreamworks_like/performance_language.md
    - style_profiles/dreamworks_like/camera_language.md
    - style_profiles/dreamworks_like/rhythm_language.md
    - style_profiles/dreamworks_like/lighting_language.md
    - style_profiles/dreamworks_like/negative_constraints.md
  fallback_policy:
    can_be_default_for_phase1_3d: false
    note: 第一阶段仅作为显式确认后的可选风格包，不作为系统默认回退项。
```

## 使用原则

- 它强调娱乐性、冲突性和角色态度，但不能退化成纯吵闹闹剧。
- 对外提示词与发布文案不直接使用品牌名。
- 风格包负责强化“怎么闹、怎么撞、怎么爆点更清楚”，不改主链结构。

## 代表作品（Representative Works）

- 《怪物史瑞克》
- 《马达加斯加》
- 《功夫熊猫》

说明：

- 这些作品只用于帮助理解“外放喜剧冲突 3D（Outward Comedic 3D）”的导演语言。
- 运行时应提炼共同的风格特征，不直接把具体作品当成复刻目标。
