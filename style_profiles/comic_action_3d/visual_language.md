# `comic_action_3d` 视觉语言（Visual Language）

## 核心方向

- 更强图形感
- 动作轨迹一眼可读
- 视觉冲击优先于细腻写实
- 画面允许更明显的漫画化设计重心

## 可直接继承的视觉锚词

- 图形感更强的 3D 动画动作电影质感
- 清晰速度线与动作方向感
- 冲击帧与姿态定格感
- 更鲜明的前后景速度层次
- Hero Moment 优先的画面设计
- comic action feature animation

## 角色设计要求

- 角色姿态必须具备动作识别度
- 轮廓要支持高速运动下的一眼可读
- 表情系统要能兼容夸张反应和高能爆点
- 服装、武器、道具要帮助建立动作路径与打击感

## 场景与道具要求

- 场景结构应有利于追逐、跳跃、对撞和转折
- 道具可以作为动作路线、打点和反击节点
- 视觉组织优先服务速度、方向和冲击
- 主场景、角色、核心道具必须支持连续动作设计
