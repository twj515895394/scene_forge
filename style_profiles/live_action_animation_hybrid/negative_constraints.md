# `live_action_animation_hybrid` 负向边界（Negative Constraints）

## 禁止项

- 不要把所有信息都依赖特效字幕表达。
- 不要让强化层显得廉价或杂乱。
- 不要牺牲真人表演的可信度。

## 谨慎项

- 当前仍是实验风格，验证不足点在于强化层很容易没有规则。
- 与 `docu_comic_hybrid` 的边界在于它更主观、更情绪化，不是纪实批注。

## 当前验证不足点

- 该风格当前仍属于 Exploration Pool，资料深度已补齐到可运行水平，但尚未经过系统性的真实项目验证。
- 使用时应优先选择该风格真正擅长的题材，避免把它当成通用风格包。
- 若后续测试发现输出与相邻风格高度重叠，应优先回到本包边界定义做收紧。
