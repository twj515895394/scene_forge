# `live_action_animation_hybrid` 表演语言（Performance Language）

## 核心方向

- 真人自然表演为主
- 动画强化放大主观感受
- 笨拙感和想象力可以并存
- 情绪节点比动作节点更重要

## 表演偏好

- 真人自然表演为主 应通过具体反应、姿态或停顿被看见。
- 动画强化放大主观感受 应通过具体反应、姿态或停顿被看见。
- 笨拙感和想象力可以并存 应通过具体反应、姿态或停顿被看见。
- 情绪节点比动作节点更重要 应通过具体反应、姿态或停顿被看见。

## 不适合的表演倾向

- 不要把所有信息都依赖特效字幕表达。
- 不要让强化层显得廉价或杂乱。
- 不要牺牲真人表演的可信度。
