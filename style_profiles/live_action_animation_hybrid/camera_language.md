# `live_action_animation_hybrid` 镜头语言（Camera Language）

## 核心方向

- 真人节奏与动画强化节点同步
- 镜头应给强化层留出落点
- 主观镜头和客观镜头切换要清楚
- 增强层不能遮蔽演员关键表情

## 镜头偏好

- 真人节奏与动画强化节点同步 应成为稳定可复用的镜头选择倾向。
- 镜头应给强化层留出落点 应成为稳定可复用的镜头选择倾向。
- 主观镜头和客观镜头切换要清楚 应成为稳定可复用的镜头选择倾向。
- 增强层不能遮蔽演员关键表情 应成为稳定可复用的镜头选择倾向。

## 节奏习惯

- 镜头切换优先服务 真人自然表演为主 与 动画强化放大主观感受 的情绪推进。
- 当项目进入高潮段时，应优先保留最能代表 `live_action_animation_hybrid` 的镜头组织，而不是借用通用快切。
- 镜头运动、停顿和转场都要帮助观众读出 真人表演+动画强化混合 的独特节拍。
