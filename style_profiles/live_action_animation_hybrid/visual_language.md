# `live_action_animation_hybrid` 视觉语言（Visual Language）

## 核心方向

- 真人基底上叠加图形化强化
- 特效批注
- 情绪可视化
- 现实与主观层叠加

## 可直接继承的视觉锚词

- 真人基底上叠加图形化强化
- 特效批注
- 情绪可视化
- 现实与主观层叠加

## 设计要求

- 角色、场景和核心道具都要服务于“真人表演+动画强化混合”的主情绪与空间秩序。
- 视觉变化应围绕 都市奇想、主观情绪 等题材展开，而不是平均化套模板。
- 同一镜头内保持一个主视觉逻辑，避免混入相邻风格的冲突语言。
- 关键画面必须让人一眼读出该风格的媒介属性和情绪基线。
