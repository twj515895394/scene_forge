# `poetic_2d_fantasy` 镜头语言（Camera Language）

## 核心方向

- 慢节奏
- 自然观察
- 静观式转场
- 让空间自己说话

## 镜头偏好

- 慢节奏 应成为稳定可复用的镜头选择倾向。
- 自然观察 应成为稳定可复用的镜头选择倾向。
- 静观式转场 应成为稳定可复用的镜头选择倾向。
- 让空间自己说话 应成为稳定可复用的镜头选择倾向。

## 节奏习惯

- 镜头切换优先服务 克制 与 体察 的情绪推进。
- 当项目进入高潮段时，应优先保留最能代表 `poetic_2d_fantasy` 的镜头组织，而不是借用通用快切。
- 镜头运动、停顿和转场都要帮助观众读出 诗意幻想手绘动画 的独特节拍。
