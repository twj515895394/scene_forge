# Open Reference Note

SceneForge templates, examples, enum values, asset patterns, and prompt fragments are starting points, not final limits.

Runtime use:

```yaml
selection_mode: reference | adapted_reference | custom_generated
source_reference:
reason:
```

Rules:

- Use `reference` when a listed option fits well.
- Use `adapted_reference` when a listed option needs changes.
- Use `custom_generated` when no listed option fits the current project.
- Unless a field says `strict_enum: true`, enum values are open examples.
- Custom choices should briefly explain why they fit the current story, character, emotion, action, shot, or sound need.

This note does not change confirmation gates, runtime path restrictions, stage contracts, or safety boundaries.
