# scene-storyboard-director

负责把剧本转成可执行分镜，强调动画电影化镜头语言，而不是复刻原片拍法。黑板记录分镜摘要与版本信息，完整分镜另行落盘。

## 何时使用

在以下场景使用此技能：

- 总控 Skill 发现项目 `next_stage` 为 `scene-storyboard-director`
- 已完成剧本改编，需要进入镜头级表达设计
- 需要把角色动作、表情、场景调度和光影节奏收成可用于后续提示词构建的分镜方案

如果剧本还没定稿，或已经进入提示词构建阶段，不要优先使用本技能。

## 执行步骤

1. 读取项目 `PROJECT_BOARD.md`，确认 `project_status` 已到 `script_ready`。
2. 读取剧本摘要、设计摘要和参考边界，确认哪些内容必须保留、哪些镜头表达必须规避。
3. 读取 `references/output-contract.md`，确认分镜字段、黑板摘要和长分镜落盘路径。
4. 生成动画电影化分镜，不直接照搬原版拍法。
5. 对每个镜头至少明确动作、表情、调度和情绪功能。
6. 将完整分镜写入 `details/shotlist_v*.md`；黑板只记录版本、镜头总览、关键镜头策略和路径引用。
7. 输出单个 YAML 补丁块；`summary` 使用中文，必要时附英文参数值。
8. 将状态推进建议交回总控 Skill，进入 `scene-video-prompt-builder`。

## 关键规则

- 分镜字段至少覆盖镜头编号、时长、景别、机位、镜头运动、角色动作、表情变化、场景调度、光影变化、情绪功能、声音设计、转场方式
- 不照搬原版拍法，要用动画电影化的表演、节奏、视角和情绪调度重构名场面
- 完整分镜落盘，黑板只保留可供提示词阶段消费的摘要信息
