## 2. 后端已开发的功能 (待手工端到端测试)

本会话中针对 `/review` 代码审查发现的 Gap，完成了以下后端接口和标准构建的开发：

1.  **文件预览路径动态接口 (`/api/artifacts`)**：
    *   在 `server.ts` 中新增了 `app.get('/api/artifacts')` 路由，调用 `@scene-forge/engine` 的 `Project.readManifest()` 动态获取并过滤当前阶段所登记的产物列表，去除了原本前端写死物理文件路径的问题。
    *   **待测试**：尚未在前端 UI 真实运行下与 WebSocket 宏操作（如 Start/Complete 产生文件后）进行端到端联调。
2.  **前端优先级自动匹配加载**：
    *   在 `App.tsx` 中对接了该接口，实现了根据产物评级评分（`final` -> `review` -> `draft` -> `preview`）自动匹配最佳预览文件路径。
3.  **PTY Bridge 沙箱退化降级机制**：
    *   实现了在 C++ `node-pty` 在沙箱或特定容器环境报错 `posix_spawnp failed` 时，自动平滑退化至系统原生 `child_process.spawn`，满足流式交互和 thought/tool 气泡解析的逻辑。
    *   **待测试**：已通过 `pty_bridge.test.ts` 自动化集成测试，仍需在真实 Web UI 环境中操作以检验其稳定性。
4.  **构建时序闭环**：
    *   在 `package.json` 的 `"test"` 脚本中强制前置 `tsc` 编译，消除了测试运行对过期 `dist/` 的隐式时序依赖。

## 3. 下一步任务与开发优先级 (Prioritized Tasks)

接下来的 Agent 应在本项目中按照以下排好的优先级，逐步补充后端接口并开展 UI 重构与风格开发：

### 🔴 P0 级：核心功能闭环与项目大厅数据打通
1.  **端到端联调测试**：启动开发服务，在真实 WebSocket 交互中验证已实现的 `/api/artifacts` 动态产物加载、 node-pty 降级原生 child_process 以及 ts 编译时序是否全数通过。
2.  **补充大厅基础 API**：
    *   `GET /api/projects`：扫描 Workspace 目录下带有 `PROJECT_STATE.json` 的子目录，汇总返回项目名、风格 ID、当前进度和修改时间以渲染大厅卡片。
    *   `POST /api/projects`：根据大厅提交的表单 `{ projectSlug, directorStyleId }` 在本地物理创建项目目录并初始化配置文件，并将该项目绑定为当前活跃项目。
3.  **大厅流转交互**：在前端渲染“独立的欢迎大厅 (Lobby)”，实现双击项目卡片/提交向导后平滑进入工作台（Workspace）的业务流转。

### 🟡 P1 级：功能增强与提示词快捷处理
1.  **补充字典与配置 API**：
    *   `GET /api/styles`：读取 `@scene-forge/engine` 注册的风格字典返回可用风格列表。
    *   `POST /api/projects/config`：接收修改后的风格 ID 等参数，覆写该项目的 `PROJECT_STATE.json` 配置文件并触发热广播。
2.  **重构工作台版面**：
    *   将 Timeline 步进轨置顶，释放左右宽度；将 Start/Validate/Complete 按钮从 Pipeline 中剥离，直接集成在聊天输入框上方（Action Dock）。
3.  **实现 Prompt 快速拷贝**：
    *   正文中检测特殊的 Prompt 块自动渲染为高亮玻璃质感“复制卡片”（带 `Copy` 键）。
    *   预览区头部集成折叠的「快捷 Prompt 提取板 (Quick Copy Dock)」实现一键打包单拷。

### 🔵 P2 级：视觉美化、主题切换与微动效抛光
1.  **双主题热切换**：根据设计文档中的 HSL 设计令牌规范，编写全套 HSL CSS 映射，支持一键切换 Cinema Studio Dark (深暗金影院风) 与 Modern Light (极简纸张灰)。
2.  **微动效打磨**：
    *   气泡对话/极客终端切换时的 **Card Flip 3D 水平翻转动画** (400ms)。
    *   点击 ValidationError `[🔍 定位]` 时，右侧正文自动滚动并将出错行**红色背景呼吸闪烁 3 次** (每次 400ms)。
    *   点击 Copy 提示词按钮时，按钮渐变为 `[✓ Copied]` 且卡片泛起绿色微光 (1.5s 后复原)。
