# 道具设计锁定卡 (Prop Design Constraints) - girls-skateboard-chase

本文件定义本项目中核心道具（炫彩滑板、运动防护装具）的视觉细节与物理状态机约束，下游各阶段必须严格继承。

## 1. 核心道具视觉设计

*   **滑板 A (女孩 A 的炫彩长板)**：
    *   **锁定路径**：核心道具新建 (`new_core_prop`)
    *   **板面涂装**：板面正面贴有亮橘色与炭黑色相间的棋盘格（Checkerboard）防滑砂纸；底面为亮橘色底漆涂装，印有白色泼墨以及黑色的“CHASE”与“开原”艺术字体标识。
    *   **配件材质**：四个轮子为鲜黄色高透光聚氨酯耐磨滑板轮；合金支架为哑光磨砂黑色材质，支架边缘带有轻微磨损和银灰色金属露底划痕。
*   **滑板 B (女孩 B 的炫彩长板)**：
    *   **锁定路径**：核心道具新建 (`new_core_prop`)
    *   **板面涂装**：板面正面贴有深蓝色与薄荷绿双拼（Split-color）防滑砂纸；底面为深蓝色底漆涂装，中央有薄荷绿与白色交错的流线速度纹路，并印有科技感“FUTURE”流线型英文字体。
    *   **配件材质**：四个轮子为半透明浅蓝色（冰蓝色）高强聚氨酯耐磨滑板轮；合金支架为闪亮的抛光钛合金银白色材质，带亮面偏光反射。
*   **防护装具 (半盔与护具)**：
    *   **锁定路径**：嵌入角色设计 (`embed_in_character_or_scene`)
    *   **半盔 A (女孩 A)**：亮白色亮漆面半盔，外侧带亮橘色与炭黑色国潮云纹印花，内衬为黑色吸汗海绵。
    *   **半盔 B (女孩 B)**：哑光冷灰色底色半盔，外侧带深蓝色与薄荷绿几何速度线条构成的科技风涂鸦。
    *   **手套与护具**：女孩 A 配备石板灰护肘护膝（亮橙扣带）与深灰色半指手套（左手配黄色掌心耐磨片）；女孩 B 配备粉白色护肘护膝（黑色碳纤盖板）与浅灰色全指手套。

## 2. 道具物理状态机 (Prop State Machines)

### 炫彩滑板物理状态机 (`prop-skateboards`)

| 状态 ID | 状态描述 | 可见视觉证据 (PBR / VFX) | 允许的交互动作 | 物理安全红线 |
| :--- | :--- | :--- | :--- | :--- |
| **`State_Ready`** | 起跑线静止就绪 | 板身呈水平悬空平放状态，轮子静止无旋转，金属支架高光清晰，防滑砂纸崭新。 | 女孩双脚踩在板面上进行碰拳或拍手互动。 | 轮子必须贴紧路面，重力平衡。 |
| **`State_Speeding`** | 高速滑降状态 | 轮毂产生高速旋转的动态重影（Radial blur），滑板支架在不平路面上产生轻微的微小高频物理颤动。 | 女孩俯身降低重心，双手往后收拢或拉扯衣服。 | 严禁板面物理穿模，保持板轮紧贴路面。 |
| **`State_Sliding`** | 弯道侧滑漂移 | 板尾侧向倾斜并滑出，防滑手套撑地路段手掌下溅起少量青烟，板尾塑料防磨头贴地拉出**金色火花粒子特效 (Spark VFX)**。后轮抱死侧滑产生少量白色胎尘。 | 女孩单手撑地、身体侧倾超过45度，滑板横向偏转。 | 侧滑物理滑动轨迹必须平滑，禁止突兀的物理顿挫。 |
| **`State_Ollie_Air`** | 飞跃障碍滞空 | 滑板板身脱离地面在空中上升，轮子由于失去摩擦力呈现悬空的高速空转（Free spin VFX），板底朝向镜头。 | 女孩微蜷双腿将板身带起，并在空中用手轻轻滑过滑板尾端（Board-tap 互动）。 | 空中滞空物理符合物理抛物线，禁止突兀的二次跳跃。 |
| **`State_Stopped`** | 终点缓冲刹停 | 滑板斜擦草地刹停，板尾的底面和防磨片区域留下深灰色斑驳擦痕与泥土颗粒，轮毂带有草叶碎屑。 | 女孩跳下板踩入草地，滑板由于惯性缓缓滑行一段后侧倾倒在草地上。 | 刹停动作自然，物理阻尼递增结算。 |

## 3. 下游各阶段硬性约束
1.  **火花特效约束**：`State_Sliding`（弯道侧滑）时产生的火花特效必须是**暖金色卡通微小粒子**，不得呈现写实暴力的电焊火花，以防破坏青春活力的调性。
2.  **磨损贴图连续性**：滑板底面的磨损程度必须随着故事节拍的推进（从 B01 的崭新到 B06 的板尾刮擦痕迹）呈线性递增，保证视觉细节的连续性。
3.  **专属道具与设计图绑定规范**：在原画与资产设计阶段，滑板 A 与半盔 A 必须与女孩 A 一同渲染并呈现在女孩 A 的角色说明书设计图中；滑板 B 与半盔 B 则必须呈现在女孩 B 的说明书中。每个女孩的服装配色、半盔涂层、滑板板面防滑砂纸以及轮毂色彩必须形成高度一致的“签名式色彩系统”（女孩 A 为亮橘/黑色系，女孩 B 为深蓝/白色系），以此强化专属道具与角色身份的排他性强绑定。
