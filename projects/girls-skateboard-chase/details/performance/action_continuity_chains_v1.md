## 2. 速度与动能递进链 (Velocity Progression)
*   **VGU-01 (0-10s) 起步加速期**：
    *   初始速度：`0 km/h`
    *   结束速度：`35 km/h`
    *   物理表现：蹬地腿部肌肉拉伸，轮子快速旋转的微小高频震动。
*   **VGU-02 (10-20s) 首弯控速与滑切期**：
    *   速度波动：`35 km/h -> 25 km/h (漂移减速) -> 30 km/h (出弯加速)`
    *   物理表现：女孩 A 擦地侧滑阻尼，后轮轻微抱死甩尾；女孩 B 顺着切线离心力滑过，出弯带正板身。
*   **VGU-03 (20-30s) 特技飞跃滞空期**：
    *   速度波动：`30 km/h -> 28 km/h (腾空阻力) -> 32 km/h (落地缓冲重归路面)`
    *   物理表现：腾空时滑轮脱地高速空转（无阻力 Free spin），落地时板桥和膝盖做垂直 Squash-stretch 弹性形变。
*   **VGU-04 (30-40s) 崎岖直道与牵引超车期**：
    *   速度波动：`32 km/h -> 35 km/h (气流减阻) -> 42 km/h (右侧爆发超车)`
    *   物理表现：路面颠簸导致板底高频震颤，风阻阻力使头发与衣摆剧烈后甩。
*   **VGU-05 (40-50s) 双星贴地死斗期 (速度高潮)**：
    *   速度波动：`42 km/h -> 30 km/h (双人强切撑地漂移阻尼)`
    *   物理表现：金属支架贴沥青产生剧烈反作用力，暖金色火花横向飞溅。
*   **VGU-06 (50-60s) 冲线与草坪阻尼刹停期**：
    *   速度波动：`30 km/h -> 45 km/h (冲刺) -> 0 km/h (斜擦草地摩擦强阻尼)`
    *   物理表现：板尾和轮子斜铲入草皮，草屑、泥土抛洒，角色受身前滚翻卸力，最终大字平躺。

## 3. 关键道具物理状态连续性 (Prop State Transitions)
1.  **滑板板底磨损**：
    *   B01 (崭新) -> B02 (板尾出现轻微磨擦痕) -> B03 (板尾轻微撞击凹痕) -> B05 (支架侧边带有摩擦剥落的高光露底划痕) -> B06 (板底印字“CHASE”和“FUTURE”带有明显的深灰色沥青刮擦与青草汁绿斑)。
2.  **防护手套磨损**：
    *   女孩 A 的左手耐磨手套掌心的聚氨酯垫片，在 B02 和 B05 摩擦后，表面形成斑驳的磨平与发黑拉丝质感。
3.  **头盔云纹与科技线条**：
    *   全片中头盔印花需严格保持完整，表面光泽度随着沾染白灰和泥土微尘线性降低，冲线后头盔外壳反射高光带有斑驳感。
