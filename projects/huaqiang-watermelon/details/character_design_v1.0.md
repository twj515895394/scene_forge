# 《华强卖瓜》皮克斯/小黄人风格角色设计说明书 (v1.0)

本文件详述了本项目两个核心角色——**华强**与**摊主**的 3D 动画电影级视觉设计。所有设计均服务于**“高饱和度、高光泽光滑3D动画风格（小黄人/皮克斯式）”**，并针对**疯狂喜剧（`exaggerated_comedy`）**动作进行适配。

---

## 1. 角色一：刘华强 (Huaqiang)

### 1.1 核心视觉定位
* **参考强度**：`high_anchor` (高锚点参考型)。保持极高的角色面部认知度与经典着装特征，同时将其卡通化重构。
* **风格意向**：身形微胖、神态沉稳，面部表情具有极强的张力反差——在说出最平静冷酷的台词时，眼神展现出无辜但挑衅的滑稽感。

### 1.2 3D 模型与体型设计
* **面部比例**：卡通化梨形圆脸，五官微聚。经典的短寸寸头（使用极细致的光滑微凸起塑料几何体表现发丝纹理）。下巴留有淡淡的青色胡茬（高漫反射渐变贴图）。
* **眼睛**：大而圆的双眼，眼眶线条圆润（类似小黄人系列中格鲁的比例）。眼球为半透明水晶玻璃材质，带有干净剔透的双重点状反射高光，能精准传递平静的“死鱼眼”和挑衅的“挑眉”神态。
* **体型**：五头身比例。微胖的上身，四肢微粗但极其灵活。经典的双手叉腰动作时，上衣皮衣会呈现极具张力的饱满挤压。

### 1.3 服饰、材质与色彩
* **外套**：亮黑色皮夹克。材质为高反射、光滑透亮的黑色亮面漆皮（具有皮克斯《玩具总动员4》巴斯光年级别的光泽感），在阳光照耀下反射出漂亮的白色边缘光。
* **内搭**：亮蓝色高饱和度圆领 T 恤，纯棉光滑质感，贴图无颗粒杂色。
* **下装**：高饱和度深灰亮面西裤加光滑黑色皮鞋（鞋头圆润巨大，高反光漆皮）。
* **头盔**：经典的白色骑行半盔。材质为极其亮白的高光烤漆塑料，带有一根半透明黑色的光滑护目镜挡风片。

### 1.4 表情融合变形 (Blendshapes) 范围
* **平静挑衅面容**：双眉微微挑起一高一低，嘴角挂着一丝无辜而笃定的半圆弧微笑。
* **挑瓜审视面容**：眼睛微微眯起成两条细缝，双瞳剧烈缩小，眉毛拧成卡通波浪线。
* **劈瓜后冷酷面容**：嘴巴微张成标准的圆润卡通形，眼神极其清澈无辜，与手上的爆裂特效形成强烈反差。

## 2. 角色二：西瓜摊老板 (The Vendor / Sheng Bing)

### 1.1 核心视觉定位
* **参考强度**：`high_anchor` (高锚点参考型)。完全复刻原版的光头、背心与金链子，但身形与五官放大数倍，显得嚣张霸道而滑稽。
* **风格意向**：身材极其壮硕、肌肉线条圆润光滑。神态极度嚣张傲慢，大眼睛里充满了市侩与自得，属于经典的“美式卡通喜剧反派”。

### 1.2 3D 模型与体型设计
* **面部比例**：圆滚滚的硕大光头（高反射率，像抛光塑料球一样反射周围环境）。宽大的扁平卡通鼻子，厚厚的光滑大嘴唇，嘴巴常闭成一条紧绷的斜线。
* **眼睛**：巨大的圆形眼睛（小黄人风格的大圆眼），瞳孔极黑。眼角下垂，眉毛粗且短，随着情绪波动会呈现夸张的拉伸与形变。
* **体型**：大猩猩比例体型。上身极其宽大，手臂粗壮如橡皮管，双腿较短，形成了滑稽的视觉重心。这种体型最适合做“秤上动手脚”的笨拙肢体表演。

### 1.3 服饰、材质与色彩
* **上装**：经典无袖纯白背心。贴图为细腻洁净的圆润布料纤维（无汗渍污垢，保持干净明亮的皮克斯动画质感）。
* **配饰**：脖子上挂着一根金灿灿、粗大的黄金项链。材质为极高反射率的抛光金属黄铜，在劈西瓜冲击波爆发时会在脖子上发生夸张的卡通惯性弹跳（带清脆金属碰撞音效）。
* **下装**：橄榄绿高饱和度短裤，配棕色光滑橡胶拖鞋，露出圆润粗壮的光滑脚趾。

### 1.4 表情融合变形 (Blendshapes) 范围
* **嚣张不耐烦**：单眼大单眼小，嘴角极度撇向一侧，鼻孔微缩，眉头向下压至眼睑。
* **阴谋得逞**：眉毛滑稽地交替上下跳动，嘴角咧开露出整齐亮白、没有牙缝的卡通大白牙。
* **西瓜爆开被震懵**：眼睛剧烈缩成两个黑色小圆点（星星眼/眩晕眼），嘴巴大张呈圆筒形，面部肌肉像波浪一样往后剧烈抖动挤压（“风洞化”形变效果）。

## 3. 角色绑定与 Slapstick（疯狂喜剧）动画支持

* **骨骼与网格控制**：
  * 两名角色的面部和肢体均支持极致的 **“挤压与拉伸 (Squash & Stretch)”** 属性。在大幅度表演（如摊主指着华强骂、华强突然俯身盯秤）时，角色肢体可产生 15%~30% 的拉伸变形，而网格体积保持恒定。
  * 摊主的脸部网格需要设置专属的“软体形变驱动骨骼”，专门用来接收西瓜浆汁爆裂时产生的卡通冲击波物理形变，实现浪花状面部形变。
