# 《华强卖瓜》角色锁定卡 (v1.0)

本文件不是长篇设计散文，而是给后续出图和出视频使用的角色锁定卡。

## 华强宇宙统一视觉语言基线

- 视觉目标：皮克斯电影级动画风格、超高质量 3D 角色动画、电影级灯光与镜头语言
- 世界形状语言：
  - 人物以方形为主、三角形为辅
  - 道具以真实轮廓为基础做轻度夸张
  - 场景以稳定方形空间为基底，并带少量有机弧线
- 统一比例策略：
  - 华强类成年男性角色默认 `6~6.5` 头身
  - 道具与场景遵循 `70% 真实 + 30% 夸张`
- 统一材质策略：
  - 比例动画化
  - 材质真实
  - 不走玩具塑料感，不走真人写真感
- 统一色彩方向：
  - 暖灰、墨黑、深绿、脏白作为主色
  - 少量高对比强调色只服务戏剧点
- 统一剪影要求：
  - 华强靠头盔、立领、宽肩、克制站姿建立识别度
  - 摊主靠低重心、链子、外放姿态建立识别度
  - 两人即使变成黑影，也要能区分身份和气场

## 角色 1：华强

- 参考强度：`medium_anchor`
- 资产策略：`new_full`
- 角色定位：平静、克制、压迫感强，但在夸张搞笑化路线里要保留一点荒诞反差。
- 风格锚词：皮克斯电影级动画风格、超高质量 3D 角色动画、电影级灯光与镜头语言
- 年龄感：中年男性
- 头身比：`6.5`
- 形状语言：`70% square + 30% triangle`
- 体型：中等偏壮，宽肩，略倒三角，不做肌肉夸张，不做玩具比例
- 脸型：偏瘦长，颧骨和下颌线要有辨识度
- 发型：短寸
- 剪影锚点：
  - 左手始终拎头盔
  - 立领夹克
  - 宽肩轮廓
  - 插袋或单手下垂的克制站姿
- 眼部策略：
  - 眼睛比真人略大约 15%
  - 不走萌眼路线
  - 眉压眼
  - 眼窝略深
  - 眼神冷静、有压迫感
- 服装轮廓：黑色夹克、深色裤装、摩托车头盔
- 设定板必须包含：
  - 单张 `16:9` 横版角色设定板
  - 正面、侧面、背面、四分之三视角
  - 平静盯视、轻微挑眉、怀疑、冷笑、压迫凝视至少五组表情
  - 站姿、走向摊位、抬手指向三组动作姿态
  - 头盔、夹克、裤装、鞋的局部细节小图
  - 中间主区放正面和四分之三视角，右侧放表情和动作，底部放局部细节
- 表情锚点：
  - 平静盯视
  - 轻微挑眉
  - 说狠话但情绪不炸裂
- 表演锚点：
  - 站姿稳
  - 出手干脆
  - 高潮段落里靠动作反差制造笑点
- 必须避免：
  - 真人演员脸复刻
  - 过度可爱化
  - 夸张到失去压迫感
  - 与统一视觉语言基线脱节，变成单独一张“写实中年男人图”

## 角色 2：摊主

- 参考强度：`medium_anchor`
- 资产策略：`new_full`
- 角色定位：市侩、粗鲁、脾气急，情绪升级速度快，是喜剧反应的主要承接者
- 风格锚词：皮克斯电影级动画风格、超高质量 3D 角色动画、电影级灯光与镜头语言
- 年龄感：中年男性
- 头身比：`6.2`
- 形状语言：`65% square + 20% circle + 15% triangle`
- 体型：壮实偏厚重，胸廓宽，重心更低
- 脸型：圆脸偏宽，五官更外放
- 发型：光头或极短发
- 剪影锚点：
  - 粗金链子
  - 背心外轮廓
  - 前倾或猛站起的动作倾向
- 眼部策略：
  - 眼睛比真人略大
  - 上火时眉毛强压
  - 情绪失控时眼神外放
- 服装轮廓：白背心、短裤、金链子
- 设定板必须包含：
  - 单张 `16:9` 横版角色设定板
  - 正面、侧面、背面、四分之三视角
  - 不耐烦、发火、怀疑、被震懵至少四组表情
  - 坐着回话、猛站起来、被冲击后失衡三组动作姿态
  - 金链子、背心、短裤、拖鞋的局部细节小图
  - 中间主区放正面和四分之三视角，右侧放表情和动作，底部放局部细节
- 表情锚点：
  - 不耐烦
  - 被质问时瞬间上火
  - 高潮时被冲击波打懵
- 表演锚点：
  - 肢体更大
  - 表情变化幅度比华强高
  - 喜剧效果主要来自失控反应
- 必须避免：
  - 低俗化、猥琐化措辞
  - 纯反派脸谱化
  - 夸张到失去“路边摊老板”身份感
  - 与华强的世界观比例和材质方向不一致

## 角色一致性要求

- 两人都采用高品质 3D 动画电影质感，但轮廓、气质和动作重心必须明显不同。
- 华强的压迫感来自克制，摊主的喜剧感来自失控。
- 后续所有故事板图和视频段落，都必须沿用这里的服装轮廓、年龄感和表演层级。
- 角色设定图必须看起来像“动画项目角色设定板”，不能看起来像真人演员定妆照或摄影棚双人海报。
- 角色设定图必须是一张信息密度足够高的 `16:9` 版式图，而不是只给两个角度的人像拼图。
- 两个角色必须共享同一个电影宇宙：同一材质逻辑、同一灯光逻辑、同一色彩基底，但轮廓和表情机制要明显区分。
