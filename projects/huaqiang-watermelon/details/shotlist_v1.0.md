# 《华强卖瓜》镜头表与时间轴 (v1.0)

## 分段规则

- 分段时长：10 秒
- 总段数：2
- 生成链路：
  - 先生成角色设定图
  - 再生成逐镜头故事板图
  - 再按分段视频 prompt 去生成视频

| 分段 | 时长 | 覆盖镜头 | 段落目标 |

| SEG02 | 10s-20s | S05-S08 | 揭穿猫腻、高潮爆发、平静离场 |

## 镜头设计原则

- 每个镜头都必须能独立生成一张故事板图。
- 每段视频都必须有明确起点、爆点和结尾状态。
- 台词、动作和音效不能悬空，必须能对应到具体镜头。
