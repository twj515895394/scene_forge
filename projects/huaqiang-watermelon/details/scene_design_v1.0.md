# 《华强卖瓜》皮克斯/小黄人风格场景与道具设计说明书 (v1.0)

本文件详述了本项目核心场景——**街角水果摊**，以及核心交互道具——**Q弹电子秤**的 3D 动画电影级视觉与交互物理设计。所有设计均服务于**“高饱和度、高光泽光滑3D动画风格（小黄人/皮克斯式）”**。

---

## 1. 场景设计：街角水果摊 (The Corner Fruit Stall)

### 1.1 核心视觉定位
* **参考强度**：`high_anchor` (高锚点参考型)。保持原版街角水果摊的标志性布局与杂乱市井感，但使用极具梦幻色彩与童话质感的三维重构。
* **色彩调性**：采用小黄人/皮克斯特有的高饱和度“糖果色”调性。
  * **主色调**：金黄的阳光、西瓜的高亮翠绿、招牌招幌的高亮橙红。
  * **背景色**：街道砖墙为温暖饱和的红褐色，天空为纯净高亮的湖蓝色。

### 1.2 空间布局与环境设计
* **水果摊棚架**：生锈破旧的遮阳棚在卡通化后，改用光滑、蓝白相间的条纹帆布顶棚（类似于干净明亮的冰淇淋餐车顶棚）。帆布边缘呈Q弹的荷叶边，在有风或冲击波时会有规律地波浪摆动。
* **陈列折叠桌**：经典的淡蓝色光滑塑料折叠桌（表面有干净的漫反射抛光）。桌上整齐地放着切开的亮红色西瓜样品、高反光塑料果盘，以及摆在正中央的 Q 弹电子秤。
* **西瓜堆放区**：在桌子下方和一侧，西瓜堆叠成完美的卡通金字塔造型。每个西瓜都呈现完美的椭球体，表面翠绿光滑，墨绿色条纹像手工绘制的一样干净整齐，带有诱人的反光油亮感。
* **街景背景**：街角的红砖墙面平整光滑，墙角有几株干净的卡通绿色小草，电线杆呈可爱的弧形微弯，挂着几根平滑的卡通电线。

### 1.3 场景光照与氛围
* **光照模式**：经典的“阳光充足、高对比度”的晴朗下午自然光。
* **光源设置**：
  * **主光源**：一束温暖、金黄色的强日光从侧上方打下，投下清晰但柔和的卡通阴影（软阴影边界）。
  * **边缘光**：在华强和西瓜的侧后方布置一盏高强度的暖金色边缘灯（Rim Light），使角色的轮廓与光滑皮衣边缘折射出美丽的亮边，极具高端三维电影质感。
  * **反射光**：街面和折叠桌对阳光的柔和反弹，把高饱和度的亮蓝色和西瓜红微微晕染在角色面部的阴影区。

## 2. 核心道具设计：Q弹电子秤 (Q-elastic Scale)

### 2.1 道具定位
* **资产分类**：`new_core_prop` (核心道具独立建卡)。
* **设计目的**：原版中电子秤是摊主“弄虚作假”的道具，也是矛盾冲突的导火索。在皮克斯风格中，我们将电子秤拟人化、动态化，使其成为一个具有高度 Slapstick（物理形变）反馈的喜剧道具。

### 2.2 视觉外观设计
* **形体特征**：矮胖的卡通复古电子秤。机身由光滑、高饱和度奶黄色防撞硬塑料构成（反射极强，像崭新的玩具模型）。电子秤底部有四个圆滚滚的黑色塑料支撑脚。
* **秤盘设计**：顶部的金属秤盘是一块高反光抛光不锈钢盘（具有镜面反射，能微微倒映出西瓜的绿色轮廓）。
* **表盘/屏幕**：超大的圆形表盘（带有复古的红色卡通物理指针），或者发着亮绿色荧光的LED数字显示屏。显示屏字样采用圆润的荧光数字，当老板在底部用手偷按时，数字会产生夸张的数字乱码跳动。

### 2.3 物理交互与 Slapstick（疯狂形变）动画机制
* **承重物理反馈**：
  * 当挑好的西瓜“砰”地落到秤盘上时，电子秤会像弹簧垫一样，整个机身向下剧烈压缩 30%（Squash），然后像皮球一样回弹弹起（Stretch），表盘指针像直升机螺旋桨一样疯狂打转，最终在一声可爱的“叮”铃声中定格。
* **劈西瓜时的冲击波物理反馈**：
  * 在刀刃落下的瞬间，西瓜在秤盘上发生爆炸，强烈的红色半透明西瓜汁气流呈环状横向扫过。
  * **秤盘震飞**：顶部的金属秤盘会像飞碟一样，带着划破空气的卡通旋转，螺旋式飞上天空，在空中翻滚三周半后“哐啷”一声砸在背景电线杆上，电线杆跟着发生夸张的卡通颤抖。
  * **机身形变**：秤盘飞走后，奶黄色的秤身会像受惊吓的人类一样，底部的四个黑色脚垫会像小脚一样紧紧抓地，秤身中央缩成极细的波浪状，表盘的玻璃面罩上甚至可以呈现卡通的裂纹与拟人化的“吃惊眼”数字显示。
