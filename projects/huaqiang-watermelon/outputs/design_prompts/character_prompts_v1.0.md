# 《华强卖瓜》角色设定图 Prompt 包 (v1.0)

以下 prompt 用于直接生成角色设定板，不是单张人物海报。目标是输出**单张 `16:9` 横版角色设定板**，并在一张图里容纳多视角、表情、动作和局部细节。

## 统一视觉语言前提

- 整个项目必须共享同一个电影宇宙，而不是“人物像动画电影、场景像普通 AI 背景、道具像商品图”。
- 统一形状语言：
  - 华强主打 `square + triangle`
  - 摊主主打 `square + circle + 少量 triangle`
  - 道具和场景也必须继承“稳定、清晰、轻度夸张”的轮廓逻辑
- 统一比例与材质：
  - 人物是动画比例，不是真人比例
  - 材质是真实电影材质，不是玩具塑料感
- 统一色彩：
  - 暖灰、墨黑、深绿、脏白为主
  - 高对比强调色只用于戏剧点
- 统一镜头感：
  - 角色设定板必须让人感觉它属于同一部动画长片，适合作为后续故事板和视频生成参考

## 角色 1：华强

**用途**：角色设定图 / 角色参考图  
**Prompt**：

> 请生成一张“角色设定板 / character sheet”，不是摄影棚人物海报。主角为中国中年男性“华强”，气场冷静克制，带压迫感，中等偏壮身材，短寸发型，脸型偏瘦长，颧骨和下颌线清晰，穿黑色夹克和深色裤装，摩托车头盔作为配件。整体采用高品质 3D 动画电影质感，角色比例略做动画化概括，但不能变成真人演员写实照，也不能变成玩具公仔。设定板需要同时包含：正面、侧面、背面、四分之三视角；三组表情（平静盯视、轻微挑眉、带压迫感的冷笑）；三组动作姿态（自然站姿、走向摊位、抬手指向）；头盔、夹克、裤装、鞋的局部细节小图；干净中性背景；清晰版式分区；轮廓清楚，服装层次明确，面部概括度适中，眼睛和五官要明显是动画角色设计而不是真人写真。
> 风格上请明确对齐：皮克斯电影级动画风格，超高质量 3D 角色动画，电影级灯光与镜头语言，feature animation character sheet。角色不是半写实真人，而是动画长片级角色设计。角色头身比为 6.5 头身，形状语言为 `70% square + 30% triangle`：宽肩、方胸、方下巴，同时眉毛与下颌线带一点锐角压迫感。请保证剪影识别度强：左手始终拎头盔，夹克领口略立，站姿克制稳固。眼睛比真人略大约 15%，但绝不萌化；要求眉压眼、眼窝略深，眼神冷静、危险、克制。表情区至少包含：平静、怀疑、轻蔑、冷笑、压迫凝视。整体色彩要纳入同一世界观：墨黑夹克、暖灰阴影、低饱和深绿反射、脏白中性背景辅助，不要跳出项目统一色系。材质方向必须是动画比例 + 真实材质：夹克有真实布料和磨损层次，头盔有清晰硬壳反光，但整体仍然是动画长片角色，而不是时尚写真。请使用单张 `16:9` 横版版式。版式建议：中间主区放最大尺寸的正面主视图和四分之三主视图；左侧窄区放侧面和背面；右上区放至少五组表情头部特写；右下区放三组动作姿态；底部窄条放头盔、夹克、裤装、鞋的局部细节和配件小图。整体要像动画项目的正式角色设定板，而不是人像拼图。

**负向约束**：

> 不要复刻真人演员脸，不要摄影棚男模照片感，不要单张站姿海报，不要玩具化身体比例，不要过度可爱化，不要夸张婴儿眼，不要廉价游戏建模感，不要低细节服装，不要只给两个角度，不要把设定板退化成写实人物宣传照。

## 角色 2：摊主

**用途**：角色设定图 / 角色参考图  
**Prompt**：

> 请生成一张“角色设定板 / character sheet”，不是人物海报。角色为中国中年水果摊老板，身材壮实厚重，圆脸偏宽，剃光头或极短发，穿白色背心、短裤，戴粗金链子，神情急躁粗鲁。整体采用高品质 3D 动画电影质感，表情反应幅度比主角更大，但仍要保持角色身份可信，不能画成真人定妆照。设定板需要同时包含：正面、侧面、背面、四分之三视角；三组表情（不耐烦、发火、被震懵）；三组动作姿态（坐着回话、猛站起来、失衡后后仰）；金链子、背心、短裤、拖鞋的局部细节小图；干净中性背景；清晰版式分区；轮廓和服装关系可读，面部概括度适中，整体明显是动画角色设计板。
> 风格上请明确对齐：皮克斯电影级动画风格，超高质量 3D 角色动画，电影级灯光与镜头语言，feature animation character sheet。角色不是半写实真人，而是动画长片级角色设计。角色头身比为 6.2 头身，形状语言为 `65% square + 20% circle + 15% triangle`：体块厚重、胸廓宽、重心偏低，同时局部眉线和嘴角带一点攻击性。请保证剪影识别度强：粗金链子、背心外轮廓、前倾或猛站起的动作倾向必须一眼可辨。眼睛比真人略大，但仍属于动画电影角色范围；上火时眉毛强压，情绪失控时眼神外放。表情区至少包含：不耐烦、发火、怀疑、被震懵。整体色彩必须服从同一世界观：脏白背心、旧金属链子、暖灰阴影、低饱和街面反射，不要变成彩色搞笑玩偶。材质方向仍然是动画比例 + 真实材质：背心有旧布料纹理，皮肤有真实次表面感，链子有重量感和旧金属反光，但整体仍归属于同一部动画长片。请使用单张 `16:9` 横版版式。版式建议：中间主区放最大尺寸的正面主视图和四分之三主视图；左侧窄区放侧面和背面；右上区放至少四组表情头部特写；右下区放三组动作姿态；底部窄条放金链子、背心、短裤、拖鞋的局部细节和配件小图。整体要像动画项目正式角色设定板，而不是两张站姿人物图。

**负向约束**：

> 不要低俗化身材夸张，不要复刻真人演员脸，不要怪异畸形，不要廉价玩具塑料感，不要廉价游戏 NPC 感，不要低细节服装，不要摄影棚写真感，不要只给两个站姿角度，不要把设定板退化成真人风双人站姿海报。

## 使用说明

- 先出完整角色设定板，再把生成结果作为后续故事板和视频的角色参考图。
- 两个角色要共用同一套光照方向、色彩基底和质感等级。
- 如果某个平台一次很难同时生成“多视角 + 表情 + 动作 + 细节区”，可以拆成：
  - 主设定板
  - 表情板
  - 动作板
  但默认目标仍然是一张完整角色设定板。
