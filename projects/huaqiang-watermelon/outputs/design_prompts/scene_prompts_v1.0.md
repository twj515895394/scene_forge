# 《华强卖瓜》场景与道具设定图 Prompt 包 (v1.0)

以下 prompt 用于直接生成环境设定板和道具设定板，不是普通背景图。目标是输出**单张 `16:9` 横版环境设定板**，并在一张图里容纳场景总览、镜头构图和局部细节。

## 统一视觉语言前提

- 场景和道具必须属于和角色同一部动画长片，而不是单独好看但脱节的背景图或产品图。
- 统一空间语言：
  - 大结构稳定清晰
  - 边角和局部带少量有机弧线
  - 空间关系服务镜头调度
- 统一道具语言：
  - `70% 真实 + 30% 夸张`
  - 每个关键道具都要有更强识别度，而不是普通商品轮廓
- 统一色彩和材质：
  - 暖灰、深绿、脏白、旧金属为主
  - 材质真实，但整体轮廓和灯光服务动画电影感

## 场景：街角水果摊

**用途**：场景设定图 / 场景参考图  
**Prompt**：

> 请生成一张“环境设定板 / environment sheet”，不是普通背景图。场景为白天的老街路边水果摊，空间布局清楚，适合角色对话、对峙和动作推进。画面中要包含摊棚、摆满西瓜的桌面、老式电子秤，以及摩托车可停靠的前方空地。整体采用皮克斯电影级动画风格、超高质量 3D 动画场景、电影级灯光与镜头语言，环境建立在写实基础上做动画长片级风格化，而不是游戏关卡图或童话插画。请使用单张 `16:9` 横版环境设定板：中间主区放场景总览主视图；左侧区放另一角度的空间视图或简化俯视布局；右上区放适合对话/对峙的中景构图；右下区放适合高潮爆发的动作构图；底部窄条放摊棚、西瓜堆、桌面、电子秤和地面材质的局部细节。光照为晴朗白天，主光从侧上方打入，色彩以暖灰街道底色、深绿西瓜和脏白/浅蓝摊位为主，整体要利于连续镜头调度。

## 核心道具：电子秤

**用途**：核心道具设定图 / 道具参考图  
**Prompt**：

> 请生成一张“道具设定板 / prop sheet”，不是单张商品图。对象是老式路边摊电子秤，机身结构结实，表盘或数字显示清晰，既适合近景拍摄，也适合承接夸张物理反馈。整体采用皮克斯电影级动画道具设计、超高质量 3D 道具、电影级材质表现。道具必须遵循 `70% 真实 + 30% 夸张`：基础仍是老式电子秤，但轮廓更干净、更有记忆点，正面识别度更强，侧面结构更利于动画镜头阅读。材质需要有旧金属、塑料壳、刻度面板的真实层次，但不要做成廉价商品渲染。请包含：正面视图、侧面视图、工作状态示意、高潮反馈状态示意，以及局部材质细节。整体要像动画长片里的正式道具设定板，而不是普通产品展示图。

**负向约束**：

> 不要科幻界面，不要纯玩具化造型，不要过度复杂外形，不要出现品牌风格名，不要把道具设定板退化成普通产品渲染图。

## 使用说明

- 先生成水果摊场景总览图，再生成电子秤单独设定图。
- 后续故事板图和视频段落必须继承这里的空间布局、色彩基底、材质逻辑与道具外观。
