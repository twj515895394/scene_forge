# 《华强卖瓜》音频提炼清单 (v1.0)

说明：本文件只是从分段视频 prompt 里提炼出来给人工配音或后期查看的速记版。最终视频生成所需的台词、音效和节奏要求，已经整合进 `outputs/video_prompts/video_prompt_segments_v1.0.md`。

## SEG01

- 环境声：街边白天环境底噪
- 动作声：摩托刹停、摘头盔、敲瓜、西瓜落秤、电子秤受力反馈
- 台词：
  - 老板，这瓜多少钱一斤？
  - 两块钱一斤。
  - 这瓜保熟吗？
  - 20斤？你这秤不够数啊。

## SEG02

- 动作声：揭穿瞬间的停顿、道具反应、高潮爆发、余波、摩托启动
- 台词：
  - 你瞧瞧，这秤底下……
  - 萨日朗！

## 节奏备注

- SEG01：稳 -> 紧
- SEG02：停 -> 爆 -> 收
