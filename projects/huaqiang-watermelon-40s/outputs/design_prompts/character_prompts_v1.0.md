## 4. 生成指南与参数建议

1.  **画布比例**：为了保证设定板的细节可读性，强制使用 **`16:9`** 或 **`3:2`**。
2.  **避免废图**：在负向 prompt (Negative Prompt) 中务必加入：
    *   `photo, photorealistic, photography, real human, text, words, label, watermark, messy background, portrait photography, ugly, deformed` (避免生成写实人像照、水印及杂乱背景)。
3.  **视觉统一**：在跑图时，建议使用相同的种子 (Seed) 或风格微调权重，保持华强、主摊主以及小弟的**粘土光影质感与头身比概括程度**完全在同一水平线上，形成完美的“三人胖瘦双煞”喜剧组合。
