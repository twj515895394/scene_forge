## 1. 彪哥 (Biao Ge) 角色说明书面板 Prompt

```text
角色设计说明书面板，3D迪士尼皮克斯动画长片风格，主角名为范德彪。
画面布局为一张专业的制作参考看板（character design board / character sheet）。
画面中央展示一个5头身的微胖中年男性角色。他有着圆角矩形的憨厚身躯，凸出的胖肚子。他穿着一件洗得褪色发白、袖口有些起毛的灰色旧西装外套，内搭一件鲜艳的大红色纯棉针织POLO衫。POLO衫的领子歪歪斜斜地立在西装外面，显得一高一低、十分松垮。他戴着一双带轻微蛛网状裂纹的金丝边蛤蟆镜，右侧镜腿缠绕着粗糙的黑色绝缘电工胶带。他右手腋下紧紧夹着一个黑色拉链皮革手包。
看板包含以下分区：
1. 【多视角模型区 (Multi-view Model)】：展示该角色的正面、3/4侧面、正侧面、背面的站姿三维模型图。
2. 【动作姿态区 (Action Poses)】：
   - 姿态一：在雪地中微曲双腿下蹲，双手五指微张呈“鹰爪挠”的搞笑防御招式。
   - 姿态二：坐在白桌布餐桌前，左右开弓大口嚼大葱和牛排，双腮圆鼓鼓。
   - 姿态三：右手两指夹着一张印有金色比特币符号“₿”与金箔算力集成电路纹路、正爆发暖金色圣光的黑色卡片向前递出，摆出傲然且略带得意的社会人大佬姿态。
3. 【表情矩阵 (Expressions)】：展示其透过蛤蟆镜向下看人的傲慢表情、说教时的撅嘴表情、吃大葱牛排时的油光满面、以及摔倒在雪地里镜片歪斜的尴尬挽尊哭丧脸。
4. 【细节原画区 (Detail Callouts)】：大红POLO衫歪斜立领特写、黑色胶带缠绕镜腿特写、生锈拉链手包特写。
高反差灯光，暖金色室内顶光与窗外冷蓝色雪景侧光对比，胡桃木木纹与西装粗糙质感，3D Pixar style character bible sheet, extremely detailed, octane render, film lighting, masterpiece.
```

## 2. 配角合集高度对照与制作面板 (Supporting Characters Line-up Sheet) Prompt

```text
配角角色设计说明书面板，3D迪士尼皮克斯动画风格，包含电影《百万英镑》餐厅场景中的所有六个配角（弗罗斯特、领班、总经理克莱门斯、老板老头、老板娘马琪、吃土豆老头）。
画面布局为一张专业的配角合照高度对照看板（character line-up design board / height scale sheet），背景有黑白色水平细网格高度刻度线。
六个配角从左至右站成一排，展示他们的头身比与高度落差：
1. 【领班 (Head Waiter)】：位于最左侧，7.5头身细长条，最拔尖的高度。身穿黑色呢燕尾服，戴单片眼镜，姿态傲慢笔挺。
2. 【弗罗斯特 (Frost)】：左二，6.5头身纸片人，高耸的肩膀与微弯的膝盖。深蓝色侍应生小马甲，歪斜小黑领结，右肩膀搭有一条洗发黄的白毛巾。
3. 【总经理克莱门斯 (Clemens)】：中央，6头身微胖英国绅士，灰色西服三件套，胸前挂金表细表链，头戴羊毛高筒礼帽，倒背双手，手戴白手套。
4. 【老板 (Owner)】：右三，5.5头身，矮胖大肚子。黑色条纹西装，大红领带，灰白乱蓬蓬的络腮胡子，鹰钩鼻。
5. 【老板娘马琪 (Maggie)】：右二，5.8头身宽胖体型，墨绿色蓬蓬裙，暗红色羊毛披肩，头顶插着彩色孔雀羽毛的高大折边帽。
6. 【吃土豆老头 (Potato Eater)】：最右侧，5.2头身，干瘪极度驼背，穿着补丁粗呢大衣，戴鸭舌帽，半月形老花镜，坐在小圆凳上。
看板的下半部分包含【表情与形变矩阵 (Expressions & Deformations Matrix)】：
- 表情一：弗罗斯特和领班轻蔑地斜眼向下看人的刻薄表情。
- 表情二（震惊形变）：亮出算力黑卡时，弗罗斯特下巴关节夸张地向下脱臼拉长垂至胸口（嘴呈深邃大黑洞），老板惊骇得打碎墨水瓶，老板娘马琪双手捂嘴、帽子被惊飞的夸张弹性形变。
- 表情三（极度谄媚）：总经理克莱门斯双手捧着黑卡眯眼哈巴狗笑，弗罗斯特呈90度折叠腰用银盘呈卡。
古典英伦茶室背景，3D Pixar style character line-up board, character sheet collection, octane render, studio volumetric lighting, cartoon physics.
```
