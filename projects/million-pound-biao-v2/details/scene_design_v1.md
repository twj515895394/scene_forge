# 场景设计设定卡 (Scene Design Constraints) - million-pound-biao-v2

本文件定义本项目 1903 古典英式高级茶室的 3D 空间架构与材质、灯光约束，所有后续阶段（分镜、视频生成提示词）必须严格执行。

## 1. 场景定位与复用关系
*   **场景名称**：1903古典英式高级茶室 (1903 Victorian Tea Room)
*   **设计动作**：直接复用 (`reuse_direct`)
*   **基准资产卡**：[victorian-tea-room.md](file:///Users/tangwujun/Documents/trae_projects/scene_forge/assets/scenes/victorian-tea-room.md)
*   **整体设计调性**：**3D 迪士尼/皮克斯动画电影风格 (3D Pixar Style)**。场景建筑线条带有轻微的概括弧度与厚重感（如粗大的窗框、微倾的门套），但其表面覆盖高度精细的材质。

## 2. 三物理分区空间架构 (Blocking Map)
茶室内部空间沿平行于相机的左右轴线划分为三个核心功能区，严禁在后续的相机调度中发生越轴（保证彪哥在右，店员在左的空间阵营对立）。

```
+-----------------------------------------------------------+
|                          街头雪景                         |
|                         (窗外风雪)                        |
+-----------------------------------------------------------+
|       Zone B             Zone A             Zone C        |
|      (左侧账台)        (中央大门/挂钟)     (右侧餐桌卡座)    |
|      老板/总经理        双开木门/大挂钟       彪哥/吃土豆老头|
|                                                           |
|     [ 柜台 / 账簿 ]                          [ 白桌布餐桌 ]  |
+-----------------------------------------------------------+
```

### 2.1 Zone A：中央大门与大挂钟区 (Entrance & Clock Zone)
*   **物理部件**：
    *   **大门**：双开式胡桃木拱形门，上方带半圆形雕花老玻璃，门框上方挂有一只黄铜风铃。
    *   **黄铜机械大挂钟**：**绝对垂直悬挂在木门的正上方**。大钟具有宽大的圆形表盘、夸张粗黑的巴洛克式指针，下方露出一根垂直悬挂的粗黄铜钟摆。
*   **动态约束**：
    *   在 B08、B09 的逼债拉扯中，挂钟的秒针走动需给予单独特写，秒针移动时伴随夸张的机件震动和重音“咔哒”声。
    *   在 B10 黑卡圣光亮起瞬间，挂钟秒针必须突然在两格中间完全静止。

### 2.2 Zone B：左侧账台柜台区 (Counter Zone)
*   **物理部件**：
    *   一件高达 1.2 米的深色胡桃木高柜台，柜台台面因长年摩擦而泛着微弱的哑光，上面摆着厚重的牛皮账簿、插有羽毛笔的墨水瓶，以及一盏常亮、散发黄色微光的小油灯。
*   **阵营划分**：
    *   此区为“英伦店家阵营”的指挥所。老板、老板娘马琪、总经理克莱门斯在此脑补、翻账簿、打算盘。

### 2.3 Zone C：右侧靠窗餐桌卡座区 (Dining Booth Zone)
*   **物理部件**：
    *   **白餐桌**：一张圆形木桌，上面覆有一张**浆得笔挺、雪白而略带粗糙棉布织纹的餐桌布**。
    *   **靠背椅**：两张高背棕色真皮钉扣椅，皮质表面有轻微龟裂。
    *   **格子窗**：大餐桌右侧是格子推拉玻璃老窗户，外面大雪纷飞，玻璃内侧挂满晶莹剔透的霜花结晶。
*   **阵营划分**：
    *   此区为“彪哥独立卡座”。彪哥在此用筷子吃牛排就大葱、大酱，吃土豆的老头坐在紧邻的卡座里。

## 3. 材质与渲染指标 (Material & Rendering)
为了在视频生成和图像渲染中得到极致质感，须重点描述以下物理表面特征：
*   **浆白桌布 (White Cotton Tablecloth)**：粗糙度 (Roughness) 配置为 0.95，无任何反射，布料经纬编织孔隙需清晰可辨。作为承接黑卡圣光的最佳白色反射体。
*   **油亮牛排 (Juicy Steak)**：牛排表面覆盖高反射的油光材质（粗糙度 0.15），肉筋和软骨带有半透明的 Q 弹胶质感，大酱质感浓稠、反光油亮。
*   **胡桃木装潢 (Walnut Woodwork)**：具有明显的木纹凹凸贴图（Normal Map），反射光线时呈现柔和的哑光红褐色。
*   **风雪结晶 (Window Frost)**：窗户玻璃上的霜花需配置冰晶折射与微小闪烁粒子。

## 4. 电影级灯光剧本 (Lighting Script)
*   **常态灯光 (B01-B09)**：
    *   采用**古典暖调局部布光 (Chiaroscuro)**。水晶吊灯自 Zone A 顶部投射出金黄色顶光，账台油灯提供 Zone B 左侧低亮暖光源，桌上的金属烛台提供 Zone C 面部暖光源。
    *   窗外风雪投射出冷蓝色的侧背光 (5600K)，打在彪哥的灰色西装和蛤蟆镜一侧，与室内的温暖暖光（2700K）形成冷暖对比，凸显其潦倒孤寂。
*   **黑卡亮起圣光 (B10)**：
    *   **强体积光 (Volumetric Gold Light)**：当黑卡亮出，自卡面爆发出一道强烈但柔和的金色辐射状体积光束，射向四方。
    *   整个 Zone C 和 Zone B 瞬间被照亮，环境环境光调高 2 档，白桌布上呈现耀眼的金色反光，弗罗斯特的白色毛巾和面部被涂上一层强烈的金色边缘光，充满戏剧神圣感。
*   **复原免单送客 (B11-B12)**：
    *   圣光收敛，灯光恢复常态，但 Zone A 大门敞开时，窗外的白色风雪反光大量涌入，在地面映射出大片白色雪地冷光。
