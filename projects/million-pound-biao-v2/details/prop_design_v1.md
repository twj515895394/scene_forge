# 道具设计与状态机 (Prop Design & State Machines) - million-pound-biao-v2

本文件定义本项目中核心道具的 3D 动画设计标准与状态变迁逻辑，下游阶段必须严格执行。

## 1. 开原维多利亚大饭店至尊算力钻石黑卡 (Victoria Hashrate Diamond Card)

### 1.1 基础材质与造型
*   **外观尺寸**：标准信用卡尺寸，但厚度为夸张 of 1.5 毫米（Q版动画化加厚）。
*   **表面材质**：极高光泽度的镜面亮黑色材质（类似于钢琴烤漆与黑曜石反射质感，反射率 0.9，粗糙度 0.05）。
*   **烫金印刷**：卡面正中央为深度凹陷的纯金箔烫印文字：“开原维多利亚大饭店·至尊算力钻石卡”。在金箔字样的周围，雕刻着复古的比特币双横杠符号“₿”和精细交错的金色算力集成电路纹路。字样具有 3D 浮雕凹凸感，边缘闪烁着极细的金色粒子。
*   **背面细节**：贴有一条带指纹污渍的白色写字条，上用黑色水笔书写狂草汉字“吴德荣”（吴总签名防伪印章，代表其为开原第一矿池算力的物理提领凭证）。

### 1.2 道具状态机 (Prop State Machine)

| 状态 ID | 状态名称 | 视觉证据 (Visible Evidence) | 允许的交互 (Allowed Interactions) | 物理/安全边界 (Safety Boundary) |
| :--- | :--- | :--- | :--- | :--- |
| **S_HIDDEN** | 隐蔽状态 | 完全不可见。隐于彪哥洗旧灰西装的内侧兜中。 | 彪哥伸手入怀的摸索动作。 | 无。 |
| **S_GLOWING** | 圣光亮卡 | 彪哥用食指和中指夹着黑卡递出。卡片爆发剧烈的暖金色强体积光，伴随粒子光屑，照亮周围 3 米的空间。挂钟秒针在此刻悬停静止。 | 彪哥两指持卡，弗罗斯特下巴脱臼颤抖，手悬在半空。 | 圣光仅为体积光渲染，不得产生实体物理冲击或热量。 |
| **S_VERIFY** | 克莱门斯核验 | 圣光熄灭。卡片被递交给总经理克莱门斯。克莱门斯手戴白手套，在柜台的小油灯下举起卡片，卡面反射油灯的微弱晃动黄光。 | 克莱门斯反复翻看，用手抚摸狂草签名；弗罗斯特、老板在旁探头。 | 严禁折叠或弯曲卡片；严禁水笔签名溶解。 |
| **S_RETURN** | 毕恭毕敬归还 | 卡片被放置在弗罗斯特颤抖的银色小托盘正中央，呈送回彪哥面前。 | 弗罗斯特单膝微曲端托盘，彪哥大喇喇伸手拈回卡片，插回腋下皮包。 | 托盘必须保持水平，卡片滑动范围限制在托盘内。 |

## 2. 炸牛排 (带软骨/肉筋 - Gristly Beef Steak)

### 2.1 基础材质与造型
*   **外观**：一块体积巨大的炸牛排，表面带有多孔的、炸至金黄焦脆的酥皮，盖着两个边缘微焦的煎蛋和数片厚切火腿。
*   **关键特征**：牛排内部纤维中夹杂着一圈**泛着白光的、极具韧性的白色牛软骨/肉筋**（视觉锚点，对应后厨为了省成本而不去软骨的抠门剧情）。
*   **辅助佐料**：白色陶盘旁边摆放着两根沾着泥点的嫩绿大葱，以及一碟浓稠、泛着油光的土色东北大酱。

### 2.2 道具状态机 (Prop State Machine)

| 状态 ID | 状态名称 | 视觉证据 (Visible Evidence) | 允许的交互 (Allowed Interactions) | 物理/安全边界 (Safety Boundary) |
| :--- | :--- | :--- | :--- | :--- |
| **S_JUICY** | 饱满诱人 | 盘中牛排完好，汤汁黏稠泛光，半透明的白色软骨部分隐约可见。大葱新鲜挺拔。 | 领班将餐盘“啪”地拍在桌上，桌布震动，刀叉盘子弹起。 | 严禁牛排滑落盘外；酱汁溅射范围限制在盘内。 |
| **S_MUNCHING** | 狂卷残云 | 牛排被彪哥的粗木筷子暴力撕扯，煎蛋流出明黄色的蛋黄。彪哥左右开弓，就着大葱嘎嘎大嚼，大酱在彪哥嘴角和盘子边缘涂抹。 | 彪哥狼吞虎咽，大葱被咬断，发出清脆的“咔嚓”声。 | 酱汁与葱花碎屑的飞溅需采用卡通粒子，避免产生恶心肮脏的写实感。 |
| **S_POLISHED** | 盘底锃亮 | 整个白盘子一贫如洗，甚至连酱汁都被彪哥用白面饼擦拭得像洗过一样锃亮，只剩下一块**嚼不烂的、带有一排牙印的坚硬白色牛软骨**被吐在盘子正中央。 | 彪哥满意拍肚子打嗝，弗罗斯特震惊吸冷气看着空盘。 | 盘底的高光反射参数在此状态下提至最高，以折射环境灯光。 |

## 3. 大葱、面条与粗陶碗 (Noodle & Clay Bowl)
*   **大葱**：长约 25 厘米，根部带泥点。生嚼时需有清脆的物理断裂网格，并伴随清脆的“咔嚓”音效。
*   **面条**：粗手擀面，浸泡在清澈带油星的汤中。
*   **粗陶碗**：灰色粗砂陶碗，碗沿有一处小缺口，带有拙朴的土气。

## 4. 辅助道具
*   **银色水壶与玻璃杯 (B03)**：
    *   银色水壶为带有精细巴洛克雕花的维多利亚复古银壶，壶口倒水时水流清亮湍急。
    *   高脚玻璃杯为古典水晶刻花玻璃杯，折射桌布的白色和水流的质感。当快速倒水满至杯口时，杯中的水面微微泛起波纹，并折射窗外透入的微弱光线。
*   **账单 (B08)**：
    *   窄长的浅黄色皮质夹子夹着一张写满英文字母和数字的账单，拍在桌上时，账单纸角剧烈抖动。
