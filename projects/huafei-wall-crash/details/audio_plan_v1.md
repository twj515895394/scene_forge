# 音频导演方案 v1

## 声音风格总控

- **主调性**：悲壮→喜剧→安静→史诗→温暖，BGM在笑点瞬间被滑稽音效"切断"
- **核心手法**：预期违背的听觉版——严肃BGM被道具音效拦腰截断
- **静默点**：Seg6全段无BGM（全片情感呼吸），Seg8穿透瞬间BGM骤停（拔电源式）
- **表现力扩展**：live_action_cinematic，不启用卡通形变音效。道具音效使用棚拍实录质感

## 9段音轨方案 + 段间桥接

### Seg1 (00:00-00:10) 悲壮还原
| 层级 | 内容 |
|------|------|
| BGM | 悲怆弦乐渐强(大提琴+小提琴)，A01-A02缓慢积蓄→A03名台词时推到最大→A04冲刺保持峰值 |
| 环境音 | 烛火噼啪微响、高窗风声 |
| 音效 | A02→A03间：心跳"咚——咚——"（低音鼓点，渐强） |
| 台词 | A03"陛下，你害得士兰好苦啊！！！！"干声+BGM全推 |
| **→Seg2桥接** | BGM在A04冲刺峰值保持，**0s硬切Seg2时B01撞击瞬间用橡皮鸭音效拦腰截断**——BGM不渐弱，是被"切掉"的 |

### Seg2 (00:10-00:20) 泡沫穿帮
| 层级 | 内容 |
|------|------|
| 音效 | B01撞击瞬间：**"砰——叽————"**（橡皮鸭被捏爆）→柱子弹性震颤余音(低频嗡，持续1s) |
| BGM | Seg1悲怆BGM被橡皮鸭音效"拦腰截断"，此后3秒完全无声 |
| 环境音 | 无（利用无声放大喜剧感） |
| 音效 | B04士兰"……啊？"后，微风轻吹便利贴纸角"沙" |
| **→Seg3桥接** | Seg2结尾B04无声→**保持无声过渡到Seg3 C01**，拍灰声打破静默。无声桥接，0s切 |

### Seg3 (00:20-00:30) 卷土重来
| 层级 | 内容 |
|------|------|
| 音效 | C01拍灰三联(布料摩擦"沙沙沙"，3下递增)→C02敲柱"咚咚咚"（空洞纸板声，3下递减） |
| BGM | 无BGM（干声段，让道具音效和表演驱动节奏） |
| 音效 | C03纸板柱倒地"啪嗒——"（干瘪纸板倒地轻响）→C04凝固3秒**完全静默** |
| 音效 | 场务吸面条"咻——"打破静默 |
| 台词 | 场务(声音越来越小):"那个……士兰老师？您这场已经拍了八遍了……"/士兰:"你先把面咽下去！" |
| **→Seg4桥接** | C04社死凝固最后0.5秒→**D01撞桌"咔嚓"瞬间打破**，0s切，静→爆 |

### Seg4 (00:30-00:40) 快剪蒙太奇
| 层级 | 内容 |
|------|------|
| BGM | 快节奏喜剧BGM切入(tempo~140bpm)，D01撞桌瞬间起 |
| 音效序列 | D01桌碎"咔嚓"→D02感应门"叮咚——欢迎光临"→D03香炉瘪"噗——"→D04绿幕脱落"哗啦——"→D05布料撕裂"嘶——" |
| 弹幕音效 | D01/D02弹幕文字飘入时配打字机"哒哒哒"（轻+快） |
| 过渡音效 | D06跳水预备:喜剧BGM→**奥运跳水BGM切入"登登登登～"**（经典奥运主题曲的喜剧变奏版，铜管+定音鼓） |
| **→Seg5桥接** | D06奥运BGM在跳水pose顶点保持→**音乐连续跨越Seg4→Seg5边界**，E01俯冲瞬间加入boing音效。音乐桥接，非切 |

### Seg5 (00:40-00:50) 蹦床弹飞
| 层级 | 内容 |
|------|------|
| BGM | 奥运BGM持续，每次弹跳加入滑稽变调(pitch bend下滑) |
| 音效 | E01地毯塌陷"噗"—E02蹦床弹簧"boing-boing"×8次，每次音高递减半音(从C5→C3)，弹1-3响亮，弹4-6渐弱，弹7-8沉闷 |
| 台词 | 士兰弹跳喊叫叠加(从高音尖叫→低音怒吼→无力的"放我下来…")，E02弹8时无声 |
| 音效 | E04趴地落地闷响→E05小鸟叫声"啾啾啾～"（卡通音效，清脆+轻快，与前面的混乱形成反差） |
| **→Seg6桥接** | E05小鸟最后一声"啾"→**0.5s无声**→F01翻身衣料摩擦声(极轻极慢)。BGM全部消失，节奏从9→1紧急复位。无声桥接+极慢动作过渡 |

### Seg6 (00:50-01:00) 短暂沉默
| 层级 | 内容 |
|------|------|
| BGM | **完全静默**——全片唯一没有BGM的段落 |
| 环境音 | F01-F02:极轻空调低频嗡声(片场环境)，远处模糊人声"泡沫柱子谁准备的…""蹦床谁放那里的…"（像来自另一个世界） |
| 音效 | F02:鸽子翅膀扑棱声→**"咕咕～"**（清脆，打破绝对静默的唯一声音） |
| 台词 | F02独白干声，无任何配乐，语气平淡，每句间自然停顿1-1.5s |
| 音效 | F03猛坐起:衣物锐利摩擦"唰——"（打破寂静的最强音） |
| **→Seg7桥接** | F03"唰"的尾声→**史诗交响BGM极弱起**(ppp,仅弦乐泛音)→G01夕阳石墙画面淡入。音频淡入+画面冷暖切换，音乐桥接空间转换 |

### Seg7 (01:00-01:10) 终极准备
| 层级 | 内容 |
|------|------|
| BGM | 史诗交响乐从ppp渐强(tempo~90bpm→110bpm):G01-G02弦乐泛音→G03铜管进入→G04定音鼓+弦乐全奏→G05推到ff但留一口气 |
| 音效 | G01手指触墙微摩擦"沙——"(极近距录音)→G03画白粉线"沙沙" |
| 环境音 | 户外微风"呼——"、远处鸟鸣 |
| 台词 | G05"陛下，我们来世再见"干声，BGM在台词时略退(mf)，让这句话有重量 |
| **→Seg8桥接** | G05台词余音+BGM在ff临界点→**H01冲刺爆发瞬间BGM全推至fff**。音乐连续+增强，非切。动势承接 |

### Seg8 (01:10-01:20) LED穿帮
| 层级 | 内容 |
|------|------|
| BGM | 史诗BGM从fff→ffff(H01-H02，推到极限)→**H03穿透瞬间突然静音（拔电源式，不是fade out，是0.1s内归零）** |
| 音效 | H03穿透瞬间:像素撕裂数字噪音"滋滋——"(高频)+扫描线蜂鸣(中频)+雪花噪点(白噪短爆)→0.5s→完全无声 |
| 环境音 | H04:片场环境音涌入——设备低频嗡声+对讲机杂音"沙沙"+空调运行声 |
| 音效 | H05导演"保一条！"→2秒静默→灯光师第一个掌声"啪、啪、啪"(孤立，中频清晰)→渐多→热烈鼓掌+口哨(高频) |
| 音效 | H05士兰昏倒:身体倒地闷响(在掌声中隐约可闻) |
| **→Seg9桥接** | H05掌声最后渐弱(reverb tail延长2s)→**温暖钢琴单音进入**(pp)→I01士兰醒来。掌声reverb尾音跨Seg边界，钢琴音桥接时间省略 |

### Seg9 (01:20-01:30) 不撞了+彩蛋
| 层级 | 内容 |
|------|------|
| BGM | I01-I03:温暖钢琴+吉他(治愈系，tempo~70bpm，松弛)，I03"切黑"后BGM渐弱至ppp→0 |
| 环境音 | I01-I03:片场收工环境(拆道具声、卷线声，远景模糊)→切黑后1s无声 |
| 音效 | I01拧瓶盖"咔"→喝水"咕"(极近距)→I03叠毯子布料摩擦→切黑 |
| BGM | I04彩蛋:喜剧拨弦乐pizzicato strings切入(轻快+调皮)，配合皇上憋笑→破功→大笑→滑下 |
| 音效 | I04憋笑声效(喉咙里压住的"嗯——")→破功"噗——"(漏气)→I05大笑(中气十足)→龙椅滑动"吱——"+身体滑落地面闷响 |
| 音效 | 对讲机"咔嗒"开关→皇上喘笑中说台词→**定格时所有声音骤停**→字幕滚屏:温暖钢琴+弦乐收尾(回到I01的治愈主题，形成闭环) |

```
Seg7: ▁▂▃▄▅▆▇ ppp→ff 史诗交响渐强
Seg8: ████████▌→0→██ ffff→拔电源→掌声渐起

```

## 段间桥接一览

| 桥接 | 类型 | 处理 |
|------|------|------|
| Seg1→Seg2 | 硬切+音效截断 | BGM被橡皮鸭"拦腰截断"，0s切 |
| Seg2→Seg3 | 无声桥接 | Seg2尾无声→Seg3头拍灰打破 |
| Seg3→Seg4 | 静爆切换 | 凝固→撞桌"咔嚓"+BGM瞬间切入 |
| Seg4→Seg5 | 音乐连续 | 奥运BGM跨段持续 |
| Seg5→Seg6 | 无声桥接+极慢 | 小鸟→0.5s→衣料摩擦，节奏复位 |
| Seg6→Seg7 | 音乐淡入 | 史诗BGM ppp起+画面冷暖切换 |
| Seg7→Seg8 | 音乐连续+增强 | 史诗BGM f→fff跨段延续 |
| Seg8→Seg9 | reverb尾音+音桥 | 掌声reverb tail→钢琴pp进入 |
