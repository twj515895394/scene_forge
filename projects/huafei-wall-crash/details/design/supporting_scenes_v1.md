| 项目 | 内容 |
|------|------|
| 功能 | B06释然和解空间 |
| 空间 | 冷宫棚拍的侧翼休息区，约4m×3m的开放区域 |
| 关键元素 | 帆布折叠导演椅（椅背印"主演·士兰"）、冰袋、500ml矿泉水瓶、折叠小桌 |
| 背景 | 有人在远处拆卸泡沫柱子和收蹦床（模糊） |
| 照明 | 片场工作灯，明亮偏暖，与冷宫的压抑形成对比 |
| 氛围 | 日常、放松、"戏散了"的感觉 |
| B06镜头 | 士兰坐椅子上，对镜头说话。4-5秒固定或极慢推镜 |
