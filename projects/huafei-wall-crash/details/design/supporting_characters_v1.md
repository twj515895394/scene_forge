# 配角轻量锁定卡 v1

## 场务小哥

| 项目 | 内容 |
|------|------|
| 身份 | 片场工作人员 |
| 年龄 | 22-28岁 |
| 外形 | 普通中国男性，短发，偏瘦。穿灰色T恤+深色工装裤，脖子上挂工作证吊牌，一只耳朵戴对讲耳机。手端一次性泡面碗+筷子 |
| 表情 | 被抓包时的尴尬/无辜："面条挂在嘴边，筷子悬在半空" |
| 出场 | 仅B02，纸板柱子倒下后暴露 |
| 功能 | 戳破悲壮的第一个"外人"，代表片场现实入侵 |
| 设计约束 | 不可与任何真实演员撞脸 |

## 导演

| 项目 | 内容 |
|------|------|
| 身份 | 剧组导演 |
| 年龄 | 40-50岁 |
| 外形 | 中年男性，微胖。戴深色鸭舌帽，穿深色导演马甲（多口袋），手持对讲机。坐帆布折叠导演椅，面前是监视器 |
| 表情 | 严肃专业——越认真越好笑。"好！这条情绪到了！保一条！" |
| 出场 | 仅B05，LED屏穿透后暴露 |
| 功能 | 把士兰的"悲剧"当作"好戏"，完成终极荒诞翻转 |
| 设计约束 | 导演标配造型，类型化即可 |

## 皇上（彩蛋角色）

| 项目 | 内容 |
|------|------|
| 身份 | 皇帝，士兰的丈夫 |
| 年龄 | 45-55岁 |
| 外形 | 中年男性，微胖体型。清代皇帝类型化形象：明黄色龙袍（简化款，无具体纹样）。**关键特征：八字胡+山羊胡（经典四郎胡子造型，参考陈建斌版雍正的面部毛发风格）**。短发辫。 |
| 表情 | B06彩蛋：从憋笑（嘴角抽搐、面部肌肉诡异扭动）→破防大笑（张嘴露齿，眼睛眯成缝）→笑得从龙椅上滑下来 |
| 出场 | 仅B06彩蛋，坐在养心殿龙椅上拿着iPad看回放 |
| 功能 | 观众的代入视角，callback原剧角色关系 |
| 设计约束 | 龙袍为类型化简化版，非特定剧集款式。四郎胡子为经典识别锚点 |
