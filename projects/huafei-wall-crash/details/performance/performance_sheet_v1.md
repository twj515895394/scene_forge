# 表演导演方案 v1

## 表演风格总控

- **performance_style**: exaggerated_comedy
- **style_family**: live_action_cinematic
- **核心原则**: 真人表演基底+喜剧节奏夸张。角色不卡通变形，但反应节奏、表情切换速度、肢体语言幅度可适度放大。悲壮越真，荒诞越强。

---

## 士兰 9段表演逐段方案

### Seg1 (00-10s) 悲壮还原
| 表演层 | 方案 |
|--------|------|
| 眼神 | 开场低垂看地面→缓慢抬起（3秒）→锁定前方柱子（瞳孔微缩）→眼眶渐红→泪盈眶 |
| 微表情 | 嘴唇颤抖（高频微颤），下巴肌肉紧绷，鼻翼微微张合（忍哭） |
| 肢体 | 跪姿→身体前倾蓄力→猛站起（裙摆滞后0.3秒）→双臂后摆冲刺 |
| 停顿 | 名台词喊完后0.5秒静止，让悲壮感"挂"在空中 |
| 呼吸 | 台词前深吸一口气（可见胸腔起伏），喊完后急促喘息 |

### Seg2 (10-20s) 泡沫穿帮
| 表演层 | 方案 |
|--------|------|
| 反应节奏 | 撞击→弹回→坐地→懵圈，整个过程1.8秒。表情切换在0.8秒内完成 |
| 表情切换 | 悲壮(撞击前0.1秒)→冲击(0.2秒)→困惑(0.5秒)→完全懵(1.0秒) |
| 眉毛 | 从紧锁（悲壮）→瞬间扬起（困惑），眉头上挑约3mm |
| 嘴型 | 从呐喊大张→碰撞后闭合→0.5秒后微张O形→"……啊？" |
| 肢体 | 弹回坐地后，双腿自然分开，手撑地面，身体后仰约15° |

### Seg3 (20-30s) 卷土重来
| 表演层 | 方案 |
|--------|------|
| 起身节奏 | 慢起——先拍灰（左肩→右肩→裙摆），3个动作约2秒 |
| 冷笑 | 嘴角右侧上挑（不对称笑），鼻腔轻"哼"，眼神斜视泡沫柱 |
| 敲柱子 | 指节敲3下，每下间隔0.3秒。敲完停顿0.5秒，满意点头 |
| 第二次冲刺 | 比第一次更自信——肩膀更挺，下巴微抬，嘴角带着"这次稳了"的笑 |
| 撞击后表情 | 自信→柱子开始倾斜时的疑惑→完全倒下后的"又来？？" |
| 场务出现后 | 嘴微张（想骂人但又体面）→嘴角抽搐（压住怒火）→"你先把面咽下去" |

### Seg4 (30-40s) 快剪蒙太奇
| 表演层 | 方案 |
|--------|------|
| 节奏 | 每个动作单元1.5-2秒，表情切换≤0.3秒，无停顿 |
| 撞桌子 | 用力过猛的表情→桌子碎时的意外→踉跄穿过时的狼狈 |
| 自动门 | 冲刺中→门开的瞬间脚下一滑→"哎？"→飞出画外 |
| 充气香炉 | 头撞→香炉瘪的"噗"→她一头栽进瘪香炉→拔出头，满脸灰 |
| 绿幕缠绕 | 撞→布脱落→被裹住→滚3圈→变成绿色布团→从里面伸出一只手挣扎 |
| 跳水预备 | 喘粗气→闭眼深呼吸→睁眼（眼神清明而决绝）→标准跳水pose |

### Seg5 (40-50s) 蹦床弹飞
| 表演层 | 方案 |
|--------|------|
| 俯冲瞬间 | 跳水姿势标准→头朝下→闭眼→嘴角一丝"终于解脱"的放松 |
| 弹跳1-3次 | 惊恐——"啊！"每次弹起嘴张得更大，眼睛瞪得更圆 |
| 弹跳4-6次 | 愤怒——"够了！""放我下来！"咬牙切齿，四肢挣扎 |
| 弹跳7-8次 | 认命——不再挣扎，四肢瘫软，表情空洞，嘴巴紧闭 |
| 落地后 | 趴地2秒→头顶小鸟特效→昏过去（眼睛变旋涡，允许轻微卡通化） |

### Seg6 (50-60s) 短暂沉默
| 表演层 | 方案 |
|--------|------|
| 睁眼 | 极慢——先睫毛微颤(1秒)→眼皮抬起一半(1秒)→完全睁开(1秒) |
| 翻身 | 缓慢翻身仰面，像翻身很累一样 |
| 独白眼神 | 看房梁→眼神失焦（不在看任何具体东西）→说到"争不到"时眼神微动 |
| 独白节奏 | 慢速，每句话间停顿1-1.5秒。语调平淡，不煽情 |
| 泪痕 | 脸上已有干涸泪痕（从Seg1残留），不新增泪水 |
| 坐起 | 从躺→猛地坐起，0.5秒完成，像触电一样。眼神瞬间清亮 |

### Seg7 (60-70s) 终极准备
| 表演层 | 方案 |
|--------|------|
| 触摸石墙 | 伸手极慢（像触摸圣物），指尖触墙→手掌贴墙→闭眼感受 |
| 微笑 | 全片第一个真正的笑——嘴角自然上扬，眼角有笑纹，眼睛微弯 |
| 画起跑线 | 蹲下→白粉画线→画完用袖子擦手，动作慢而仪式化 |
| 拉伸 | 压腿(2秒)、转肩(2秒)、闭眼深呼吸(2秒)——极度认真反而好笑 |
| 起跑预备 | 蹲姿，双手撑地，头低→抬头→眼神锁定50米外的石墙 |

### Seg8 (70-80s) LED穿帮
| 表演层 | 方案 |
|--------|------|
| 起跑 | 爆发力最大化——蹬地瞬间身体前倾角>30°，表情极度专注 |
| 奔跑中 | 面部肌肉被"风"吹动，嘴微张呼吸，眼神始终锁死前方 |
| 闭眼 | 撞前0.2秒闭眼，面部肌肉松弛——她在迎接死亡 |
| 穿透后踉跄 | 没有撞击→惯性让她前冲几步→踉跄停住→睁眼→表情：完全空白 |
| 环顾 | 缓慢转头180°环顾片场——灯光→摄像机→监视器→众人 |
| 听到"保一条" | 嘴角微动（想说"什么"但发不出声）→眼神从困惑→恍然→彻底崩 |
| 再次昏倒 | 眼睛上翻→身体像断电一样前倾→脸朝下倒地（标准"大"字趴） |

### Seg9 (80-90s) 不撞了
| 表演层 | 方案 |
|--------|------|
| 醒来 | 睫毛微颤→睁眼→看到冰袋→摸了一下→"哦，又活了"的平淡表情 |
| 喝水 | 拧瓶盖（普通动作）→喝一小口→放瓶→看片场（长凝视） |
| 独白节奏 | 每句话之间0.8-1秒自然停顿，像和老朋友聊天 |
| 微笑 | "那就凑合活吧"时嘴角微扬，不夸张，自然地释然 |
| 回头说道具预算 | 边走边回头，说"上点心"时挑了挑眉毛（唯一一次活泼） |
| 对镜头 | 整段面向镜头或镜头方向，打破第四面墙 |

## 配角表演方案

### 场务小哥 (Seg3)
| 表演层 | 方案 |
|--------|------|
| 被抓包瞬间 | 身体微微后缩→筷子悬在半空→面条挂在嘴边→嘴微张→眼睛圆睁 |
| 沉默3秒 | 与士兰对视→眼神游移（不敢直视）→吸面条（慢动作，面条"咻"地吸进嘴） |
| 说话 | 声音越来越小："那个……士兰老师？您这场已经拍了八遍了……" |
| 扶柱子 | 放下泡面→站起来→弯腰扶柱子，动作小心翼翼，像做错事的孩子 |

### 导演 (Seg8)
| 表演层 | 方案 |
|--------|------|
| 初看 | 盯监视器→突然身体前倾→摘下帽子→凑近屏幕 |
| "好！" | 拍大腿→站起→竖起大拇指→眼睛发光——"这条情绪到了！" |
| "保一条" | 职业习惯式挥手，语气理所当然，完全没意识到士兰已经昏倒 |
| 困惑 | 副导凑近耳语→导演歪头→"穿到B组了？"→若有所思→耸肩（不在乎） |

### 皇上 (Seg9) 彩蛋
| 表演层 | 方案 |
|--------|------|
| 憋笑四阶段 | 每个阶段约2秒，连续无cut |
| 阶段1 | 嘴唇紧抿→腮帮像松鼠一样鼓起→眼角肌肉开始抽动→鼻孔微张 |
| 阶段2 | "噗——"一声漏气→嘴角一侧失控上翘→赶紧用手捂住嘴→肩膀抖动 |
| 阶段3 | 手松开→张嘴大笑→露出上下牙齿→眼睛眯成两条缝→头后仰→iPad晃 |
| 阶段4 | 身体重心偏移→一只手抓龙椅扶手→整个人慢慢滑下去→龙袍在地面皱成一团 |
| 擦泪 | 笑到流泪→用袖口擦眼角→拿起对讲机→说台词时还在喘 |

## Hero Moment 表演重点

### H01: LED穿透 (Seg8 SH15)
- 全片唯一使用超慢镜的表演段落
- 士兰闭眼到穿透的0.1秒：面部肌肉从紧绷→完全放松
- 无声处理——这段没有台词、没有音效，只有视觉

### H02: 地上独白 (Seg6 SH11)
- 极俯拍+极慢节奏，士兰独白速度比正常台词慢40%
- "争不到"之后1.5秒完全静止——让这句话沉下去

## 动作连续性链

```
Seg1 跪→站→跑（向北） → 
Seg2 撞→弹→坐 → 
Seg3 站→敲→撞→柱子倒 → 
Seg4 撞桌→撞门→撞香炉→撞绿幕→跳水预备（向南俯冲） → 
Seg5 跳→弹飞8次→趴地 → 
Seg6 翻身→躺→坐起（动作由慢→突变快） → 
Seg7 站→走→触摸→蹲→拉伸→起跑预备 → 
Seg8 跑→撞→穿透→踉跄→昏倒 → 
Seg9 醒→坐→喝水→站→走→回头
```

```
暴躁→崩溃(Seg4) → 认命(Seg5) → 
心酸→觉醒(Seg6) → 
希望→仪式感(Seg7) → 
终极荒诞→崩塌(Seg8) → 
释然→温暖(Seg9)
```
