# 角色说明书：士兰 v1

## 身份档案

| 项目 | 内容 |
|------|------|
| 角色名 | 士兰 |
| 身份 | 皇帝的妃子 |
| 年龄 | 30-35岁 |
| 气质 | 高傲→悲愤→暴躁→崩溃→心酸→释然 |
| 故事功能 | 主角，灾难承受者与最终和解者 |
| 风格家族 | live_action_cinematic / classic_studio_wuxia |

## 外形描述

### 脸型（参考蒋欣面部特征）
- 圆润鹅蛋脸型，面颊饱满但不臃肿
- 下颌线柔和但轮廓清晰，下巴微尖
- 颧骨不过分突出，整体给人华贵端庄感
- 眉骨适中，眉眼间距舒展
- 面部的"贵气"来自五官比例而非尖锐线条

### 五官
- **眼睛**：杏眼，眼尾微微上挑，双眼皮清晰。眼神从高傲凌厉→悲愤含泪→崩溃空洞→平静释然，是表演的核心载体
- **眉毛**：细长柳叶眉，眉尾略挑，符合清代妃子妆容特征
- **鼻子**：鼻梁挺直但不尖锐，鼻翼适中
- **嘴唇**：唇形饱满，嘴角在冷宫阶段微向下（悲愤），结局阶段微向上（释然）
- **肤色**：白皙，冷宫阶段略显憔悴

### 发型
- 清代两把头（简化版，非具体剧集复刻款式）
- 发髻适中大小，不过分夸张
- 冷宫阶段发丝略有散落（悲壮感）
- 头饰简化：素银簪或木质簪，不用具体剧集的金饰款式

### 服装
- **B01-B05（冷宫阶段）**：素白/浅灰旗装，面料为棉麻质感，略有褶皱。袖口和领口有极简镶边，不用具体纹样。整体体现"被废黜的妃子"的潦倒
- **B06（结局阶段）**：同一套旗装，但肩上披着剧组毯子（片场道具），头上戴着冰袋

### 身材比例
- 身高约168cm，修长身形
- 旗装廓形：宽袖、直筒，不强调腰线

## 表情系统

### 剧情表情（6个）
1. **悲愤呐喊**：仰头，嘴大张，眼眶通红含泪 —— B01名台词"陛下，你害得士兰好苦啊！！！"
2. **懵圈困惑**：坐在地上，眉毛扬起，嘴微张，眼神空洞 —— B01撞到泡沫柱子后
3. **冷笑不服**：嘴角一侧上挑，眼神不屑，下巴微抬 —— B02"区区泡沫，也配挡本宫？"
4. **暴躁崩溃**：披头散发，咬牙切齿，眉头紧锁，鼻孔微张 —— B03连环失败
5. **心酸独白**：躺地望天，眼神空洞而平静，嘴角微微下垂，泪痕未干 —— B04唯一正经时刻
6. **释然微笑**：对镜头淡淡一笑，眼神温暖而疲惫 —— B06"不撞了，凑合活吧"

### 微表情（4个）
1. **迟疑皱眉**：眉头微蹙，眼神飘向一侧 —— B02敲柱子判断材质
2. **震惊瞪眼**：眼睛瞬间睁大，嘴唇抿紧 —— B05穿透LED墙后
3. **嫌弃撇嘴**：下唇微微外翻，眼睛半眯 —— B03绿幕布缠绕
4. **喝水放松**：眼神放空，嘴唇轻抿瓶口 —— B06结局

## 动作姿态（6个剧情动作）
1. **悲壮冲刺**：身体前倾，双臂后摆，头微微低下，冲向柱子/墙
2. **坐地懵圈**：双腿分开坐在地上，手撑地面，身体后仰
3. **起跑预备**：蹲姿，一手撑地，像田径运动员起跑 —— B03蹦床段落
4. **弹飞空中**：四肢张开，表情惊恐，身体在空中不受控制 —— B03蹦床弹跳
5. **躺地平复**：仰面朝天，手臂自然摊开 —— B04独白
6. **椅上休息**：靠坐帆布折叠椅，一手拿矿泉水瓶，毯子披肩 —— B06

## 关键道具交互
- **泡沫柱子撞击**：身体撞向柱子→柱子弹性变形→人被弹回坐地
- **矿泉水瓶**：拧开、喝水、放在腿上 —— B06日常化动作，体现释然

## 比例对照
- 士兰 vs 场务小哥：士兰略高（旗装+旗鞋），场务为普通现代人身高
- 士兰 vs 导演：导演坐在监视器前，身材中等偏壮
- 士兰 vs 皇上：皇上坐龙椅上，士兰站立，基本等高（不穿旗鞋时）

## 物理与安全边界
- 所有撞击不产生写实伤害：无流血、无骨折
- B01泡沫柱弹回：身体后坐，无伤痕
- B03蹦床弹飞：四肢在空中滑稽摆动，落地脸朝下但无伤（可加头顶金色小鸟转圈卡通特效）
- B04躺地：脸上有灰渍但无伤口
- B05穿透LED：无物理碰撞，直接穿过
- B06：头上有冰袋（卡通化处理，非写实伤包）

## 表现力扩展边界
- 当前 style_family = live_action_cinematic，不启用卡通形变动画
- 喜剧效果通过棚拍道具的"物理穿帮"实现，而非角色身体变形
- 允许：轻灰脸、头戴冰袋、衣服褶皱脏污
- 禁止：写实流血、骨折、身体恐怖、持续痛苦表情
