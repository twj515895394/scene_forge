# 场景设定：冷宫/棚拍片场 v1

## 场景身份

| 项目 | 内容 |
|------|------|
| 场景名 | 冷宫/棚拍片场 |
| 故事功能 | B01-B05主场景，从"真实冷宫"渐进暴露为"棚拍片场" |
| 风格家族 | live_action_cinematic / classic_studio_wuxia |
| 美学核心 | 棚拍布景感——场景本身是最大的喜剧道具 |

## 双状态设计

### 状态A：冷宫（B01开场）

**空间布局**
- 长方形室内空间，约8m×6m
- 正后方：一面看起来真实的石墙（实际是LED屏）
- 左侧：两根看起来像石头的柱子：前柱(泡沫)和后柱(纸板)，间距约2m
- 右侧：破旧木桌一张，上面放香炉
- 地面：深色地毯覆盖（下方藏蹦床）
- 光源：几盏烛台，烛火摇曳

**材质与细节**
- 墙面：暗灰色，有斑驳痕迹和假蛛网
- 柱子：仿石纹理涂装，但在B01撞击时暴露泡沫/纸板真相
- 地面地毯：深红色，边缘磨损
- 窗户：高窗，透入微弱月光
- 氛围：压抑、破败、孤绝

**棚拍式灯光**
- 主光：烛火暖光（实际为棚内柔光灯模拟）
- 轮廓光：微弱的月光从高窗洒入
- 整体照度偏低，强调幽闭感

### 状态B：棚拍片场（B02-B05渐进暴露）

**逐步暴露的元素**
- B02：纸板柱子倒下→露出背后的场务小哥和工作推车
- B03：绿幕布被扯下→露出后面的灯光架；地毯掀起→露出蹦床
- B05：士兰穿透LED墙→完全进入片场区域

**片场区域（Zone D，B05完全揭示）**
- 导演监视器+折叠椅
- 摄像机轨道
- 灯光架：LED面板灯、柔光箱
- 收音麦克风吊杆
- 工作推车：上面放道具、剧本、对讲机
- 地上有彩色胶带标记T形站位点
- 整体照明：冷白LED工作灯+暖色柔光灯混合

## 镜头朝向与轴线
- 默认主镜头朝向：从Zone D（片场后方）朝向Zone A（中心表演区），即从南向北
- 180度轴线：东西方向，士兰的冲刺路径沿南北轴线（从Zone A冲向后墙）
- 次级角度：B05穿透后反向镜头，从后墙朝Zone D拍摄

## 空间锚点
1. **泡沫柱子**：Zone B左侧，始终可见，是空间定位的"北极星"
2. **后墙石墙**：正后方，B05的视觉目标
3. **导演监视器**：Zone D，B05揭示的"真相"
4. **高窗月光**：天花板位置，B01-B04始终存在，提供空间连续性
