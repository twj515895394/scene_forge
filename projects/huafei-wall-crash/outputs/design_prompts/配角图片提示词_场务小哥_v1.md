# 配角图片提示词：场务小哥 v1

## 角色设定图 Prompt

```
电影级角色设定图，经典棚拍武侠电影感，真人电影质感。

中国年轻男性，22-28岁，短发，偏瘦身材。
圆脸，五官普通，不引人注目——典型的"片场工作人员"面貌。
穿灰色短袖T恤，深色工装裤（多口袋），脖子上挂蓝色工作证吊牌，
一只耳朵戴对讲机耳机，黑色耳机线垂在胸前。
双手端着一个一次性纸碗泡面，右手拿一次性竹筷，筷子上夹着几根面条。
面条挂在嘴角边，表情尴尬无辜——"被抓包了"的表情。
眉毛微扬，眼睛睁大但不敢直视，嘴唇微张。

背景：模糊的片场环境（金色暖光+器材剪影）。

半身中景，正面略偏3/4。
棚拍式柔光，面部轮廓光清晰。
```

## 表情变体

```
同一角色，三种表情并排：
1. 尴尬无辜（主表情）：面条挂嘴边，眼睛圆睁，眉毛扬起
2. 慌乱心虚：眼神飘向一侧，嘴角抽搐，泡面碗微微倾斜
3. 职业假笑：嘴角僵硬上扬，眼神空洞，标准"打工人"微笑
```
