# 场景设定图提示词：冷宫/棚拍片场 v1

本提示词用于生成冷宫/棚拍片场的场景设定图与场景资产总参考图。

---

## 主场景图 Prompt（冷宫正常状态 / State A）

```
电影级场景设定图（Film Set Design Reference），经典棚拍武侠电影感。

清代冷宫内景，长方形空间约8m×6m。
正后方：一面看起来真实的暗灰色石墙，表面有斑驳痕迹和蛛网。
左侧：两根石柱（仿石纹理涂装），间距约2m，前柱位置贴有一张黄色便利贴。
右侧：一张破旧深色木桌，上放一只铜质香炉。
地面：深红色磨损地毯，覆盖整个地面。
天花板左侧：一扇高窗，透入微弱青白月光。

照明：棚拍式轮廓光，烛台暖光为主光源，月光为辅助轮廓光。
整体照度偏低，强调幽闭压抑的氛围。

材质：木质纹理、石材纹理（实际为泡沫和纸板涂装，但外观逼真）、棉质地毯。
气氛：破败、孤绝、昏暗，带有一丝不自然——过于对称的构图暗示这是布景。

视角：广角平视，从场景前方（南侧）看向后墙（北侧）。
拍摄设备：电影摄影机视角，浅景深，前景略虚化。
包含摄像机轨道、灯光架、麦克风吊杆等棚拍设备在画面最前景或边缘（模糊处理）。
```

## 场景状态对比图（双状态并排）

```
左右并排双场景设定图。

左图：冷宫正常状态（同上方主场景图）。

右图：冷宫棚拍片场暴露状态。
- 同一空间，但纸板柱子已倒在地面，露出后面的工作推车
- 地面深红地毯一角掀开，可见黑色蹦床金属边框和网面
- 后墙绿幕布被扯下半幅，露出后面的灯光架（LED面板灯+柔光箱）
- 场景右侧可见导演监视器和折叠椅
- 地上有彩色胶带标记的T形站位点
- 照明变为冷白LED工作灯与暖色柔光灯混合
- 整体：从"古装冷宫"暴露为"现代摄影棚"
```

## 负向约束
- 不要写实血迹、尸骸、刑具（冷宫不是牢房）
- 不要特定剧集标志（翊坤宫牌匾、具体剧名）
- 不要现代科技物品出现在冷宫状态（保留给棚拍状态）
- 不要过度恐怖或阴森（不是恐怖片场景）
- 冷宫状态和棚拍状态的空间结构必须一致（同一空间的不同揭示阶段）
