# 角色说明书图片提示词：士兰 v1

本提示词用于在 AI 图像生成平台生成 **士兰（皇帝的妃子）角色说明书板 / Character Design Bible Sheet**。
目标输出为多分区角色设计资料板，非单张海报。

---

## 角色说明书板整体要求

```
角色设计说明书板（Character Design Bible Sheet），电影级制作资料板布局。
板式要求：
- 板面分为多个清晰分区，每个分区带中文标签
- 中央：角色正视图（全身），站立姿态
- 左上：3/4侧面头像（中景）
- 右上：侧面头像（中景）
- 左下：背面视图（全身）
- 下方从左到右：6格表情阵列 + 4格微表情阵列
- 右侧纵向：4个关键动作姿态
- 右下角：细节区（手部+头发+服装领口特写）
- 最下方：比例对照条（士兰 vs 场务小哥 身高对比）
- 板面背景：中性灰，带细线网格参考
- 整体为棚拍电影感角色资料板，非漫画非动画风格

角色名称标签：中文"士兰"在板面顶部居中，下方小字"皇帝的妃子 / Imperial Consort"
```

## 角色外形

### 脸型与五官（参考蒋欣面部特征）
```
圆润鹅蛋脸型，面颊饱满柔和但不臃肿，下颌线条清晰柔和，下巴微尖。
颧骨不过分突出，展现华贵端庄的气质。
杏眼，眼尾微微上挑，双眼皮清晰，眼神高傲中带悲凉。
细长柳叶眉，眉尾略挑，符合清代宫廷妆容特征。
鼻梁挺直但不尖锐，鼻翼适中。
嘴唇饱满，唇色偏淡，嘴角自然微向下，带一丝悲苦。
肤色白皙，冷宫状态略显憔悴但仍有贵气残留。
面部特征高度辨识度：面部"贵气"来自五官比例的端庄而非线条的尖锐。
```

### 发型与头饰
```
清代两把头（简化版，非特定剧集复刻）。
发髻大小适中，不夸张。
黑色发丝梳理整洁但冷宫阶段有几缕散落发丝（增添悲壮感）。
头饰极简：一支素银扁方簪横贯发髻，无其他华丽饰品。
```

### 服装（冷宫素白旗装）
```
素白色/浅灰调棉麻质感旗装长袍。
面料有自然的细微褶皱，体现冷宫潦倒状态。
宽袖，袖口有极简灰色镶边（无花纹，仅单色滚边）。
领口为小立领，同样极简滚边。
直筒廓形，不强调腰线，符合清代旗装基本制式。
整体去品牌化：无任何特定剧集纹样、刺绣或配色复刻。
```

### 身材
```
身高约168cm，修长身形。
旗装廓形宽大，不紧身。
脚穿素色平底布鞋（冷宫不使用旗鞋/花盆底）。
```

## 表情系统

### 6个剧情表情（按表演时间排序）
1. **悲愤呐喊**：仰头15°，嘴大张，眼眶发红含泪光，眉头紧锁，面部肌肉紧绷——B01高潮表演
2. **懵圈困惑**：眉毛高高扬起，眼睛圆睁，嘴微张呈O形，头部微向前倾——B01撞泡沫柱后
3. **冷笑不服**：嘴角一侧上挑（歪嘴笑），眼神斜视不屑，下巴微抬——B02"区区泡沫，也配挡本宫"
4. **暴躁崩溃**：披头散发（发丝散落更多），咬牙切齿，眉头深锁，鼻孔微张，双眼冒火——B03连环失败
5. **心酸平静**：平视前方，眼神空洞而平静，嘴角微微下垂，泪痕可见但已干——B04独白
6. **释然微笑**：对镜头淡淡微笑，嘴角自然上扬但幅度不大，眼神温暖而放松——B06"不撞了，凑合活吧"

### 4个微表情
1. **迟疑皱眉**：眉头微蹙，眼神飘向右上方（思考状）——B02敲柱子判断材质
2. **震惊瞪眼**：眼睛瞬间睁大至极限，嘴唇抿紧成一条线——B05穿透LED墙瞬间
3. **嫌弃撇嘴**：下唇微微外翻，眼睛半眯——B03被绿幕缠绕
4. **喝水放松**：眼神放空望向远方，嘴唇轻抿矿泉水瓶口——B06结局

## 动作姿态（4个关键姿态）
1. **悲壮冲刺姿**：身体前倾约30°，双臂向后摆，头微低，左脚前右脚后，裙摆飘起——奔跑中
2. **坐地懵圈姿**：双腿分开坐在地上，手撑身后地面，身体后仰15°，表情懵圈——撞后
3. **弹飞空中姿**：四肢大张成X形，裙摆向上翻飞，表情惊恐，身体在空中悬停——蹦床弹飞
4. **椅上放松姿**：靠坐帆布折叠椅，背上披毯子，一手垂放一手拿矿泉水瓶搁腿上——B06

## 细节区
- **手部特写**：纤长手指，指甲干净无装饰，手背可见细微青筋，一只手持矿泉水瓶
- **发髻背面**：展示两把头后侧结构（简化版发髻的盘发方式）
- **领口细节**：小立领+极简灰色滚边，领口内侧可见衬里

## 负向约束（Negative Constraints）
- 不要写实流血、伤口、身体恐怖
- 不要特定剧集旗装花纹、刺绣、头饰款式
- 不要现代服装元素
- 不要过度华丽的金饰或珠宝
- 不要背景中的具体剧集标志（翊坤宫、甄嬛传相关文字/图案）
- 不要卡通变形或动画风格
- 无水印、无文字遮挡角色主体
- 角色说明书板不要退化为单张站姿人像
