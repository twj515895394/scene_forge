# 空间站位图提示词 v1

本提示词用于生成冷宫/棚拍片场的空间站位参考图（Space Blocking Reference），
供后续故事板分镜和视频提示词阶段继承空间关系。

---

## 俯视空间站位图（Top-Down Blocking Map）

```
技术图纸风格俯视空间站位图（Top-Down Space Blocking Reference Map）。

长方形室内空间，上方为北，下方为南。
图纸风格：白底黑色细线，类似建筑平面图但更简洁。
每个Zone用虚线框标出，标注中文标签。

空间分区：
- Zone A【中心表演区】：中央椭圆形区域，标注"士兰：起跑/冲刺/撞击"。允许移动方向箭头指向北（后墙）。
- Zone B左【泡沫柱】：左侧圆形标记，标注"泡沫柱子 | 固定位，不移动"。
- Zone B右【纸板柱】：左侧后方圆形标记（虚线，表示可倒），标注"纸板柱子 | B02触发倒下"。
- Zone C【蹦床区】：中央偏前矩形虚线框，标注"蹦床（地毯覆盖）| B03触发"。
- Zone C右【木桌区】：右侧矩形标记，标注"木桌+香炉"。
- Zone D【片场区】：底部横条区域，标注"导演监视器+摄像机轨道 | B05前不可见"。

标注：
- 红色虚线箭头：士兰冲刺路径（Zone A → 后墙 / LED屏）
- 蓝色标记：主摄像机位（Zone D前方，朝向北）
- 绿色标记：入口（西侧拱门）
- 黄色标记：出口/穿透方向（LED屏 → Zone D）

包含图例：
- 实心圆 = 固定道具
- 虚线圆 = 可变/可倒道具
- 箭头 = 角色移动路径
- 眼睛图标 = 摄像机位

比例尺：1格 = 1m
方向指示：↑北（后墙/LED屏） ↓南（片场/摄像机）

底部图例栏：中文标注各符号含义
```

## 空间约束说明（非图片内容，供下游读取）

- **士兰默认位置**：Zone A 中央，B01从西侧拱门入场后进入
- **场务小哥默认位置**：Zone D（片场后方），仅在B02纸板柱倒下后可见
- **导演默认位置**：Zone D 监视器前，B05揭示
- **禁止角色进入区域**：LED屏穿透后不可返回冷宫侧（单向）
- **允许移动范围**：士兰可在Zone A/B/C内自由移动；场务可在Zone D内移动

## 负向约束
- 不要3D渲染效果（保持平面技术图风格）
- 不要照片级真实（保持图纸/蓝图感）
- 不要复杂的建筑细节（只需空间关系和位置标记）
