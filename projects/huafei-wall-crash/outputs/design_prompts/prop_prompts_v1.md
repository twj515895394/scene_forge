## 1. 泡沫柱子

```
道具设定图，电影道具风格，摄影棚实物质感。

一根高约2.5米、直径约40厘米的圆柱。
表面涂装仿石纹理——灰色基调，有深灰色不规则纹理，远看像真石柱。
近看：柱面上贴着一张黄色正方形便利贴，手写黑色字迹：
"道具泡沫柱，请勿用力撞击——横店影视城"
便利贴一角微微翘起。

材质：高密度泡沫，表面经过仿石喷涂处理。
底部：圆形平底，无固定螺丝——说明它是独立放置的。
上端：与天花板有约5cm缝隙——暗示它是道具而非建筑构件。

侧面角度展示，棚拍工作灯光照明，中性灰背景。
柱子旁边放一把卷尺作为比例参照（显示2.5m高度）。

状态对比（左右并排）：
左：正常态——立柱笔直，仿石纹理完整
右：触发态——柱身微微向后倾斜约15°，表面出现一道浅浅的弧形压痕（撞击痕迹）
```

## 2. 纸板柱子

```
道具设定图，电影道具风格，摄影棚实物质感。

外观与泡沫柱子几乎一模一样——同样高约2.5米，同样的仿石纹理涂装。
但关键差异：
- 柱身更轻更薄，表面纹理是印上去的（非喷涂），近看有明显印刷网点
- 底部没有固定，仅靠重力直立
- 背面（设定图需展示）：中空结构，瓦楞纸板材质清晰可见，内部有木条骨架支撑

角度：正面外观 + 背面剖开视图（左右并排）。
左：正面——看起来像真石柱（但近看印刷网点暴露）
右：背面——中空纸板结构，瓦楞纸纹理，几根细木条做骨架

状态对比（下方小图）：
左：直立态（CB-01）
中：场务从背面探出头（CB-03——柱子倒下后，场务从背面空间出现）
右：柱子平躺在地面特写，展示其极轻的纸板材质

棚拍工作灯光，中性灰背景。
比例参照：柱子旁站一个1.75m的场务小哥剪影。
```

## 3. 蹦床地板

```
道具设定图，电影道具风格，摄影棚实物质感。

道具主体：直径约1.8米的圆形蹦床，黑色金属边框，黑色弹力网面。
隐藏态：蹦床被一张深红色地毯完全覆盖。地毯为棉麻材质，边缘磨损。
地毯四角用胶带固定在地面上（胶带特写可见）。

触发态（下方展示）：地毯中央塌陷，露出下方的黑色蹦床网面和金属弹簧。

角度：俯视（Top-Down）展示地毯覆盖态 + 侧面剖视图展示蹦床结构（地毯/蹦床网面/金属边框/弹簧/地面间隙）。

剖视图标注：
- 地毯层（深红棉麻）
- 蹦床网面（黑色弹力网）
- 金属弹簧圈（银色螺旋弹簧×12）
- 地面间隙（约30cm）
- 标注文字："弹跳高度约1.5m递减8次"

棚拍工作灯光，中性灰背景。
```

## 4. 绿幕背景布

```
道具设定图，电影道具风格，摄影棚实物质感。

道具主体：一块宽约3米、高约2.5米的鲜绿色幕布（chroma key green）。
上沿通过一排金属夹子固定在横杆上，横杆两端架在灯光架上。
布料为棉质，表面有细微纹理但整体平整。

触发态（下方展示）：
- 幕布上沿一侧的夹子脱落，布面产生大量斜向褶皱
- 布面一角拖到地面，形成一个绿色布堆
- 布堆中隐约可见一个人形轮廓（士兰被缠绕的示意）

角度：正面展示平整态 + 侧面展示脱落态。

标注：
- "平整态（GS-01）"
- "触发态（GS-03）：角色撞击后幕布脱落缠绕"
- "安全：不缠绕颈部，可自行挣脱"

棚拍工作灯光，中性灰背景。
```

## 5. LED显示屏墙面

```
道具设定图，电影道具风格，摄影棚实物质感。

道具主体：一面宽约4米、高约3米的大型LED显示屏墙面。
正面（朝向冷宫侧）：屏幕显示高精度仿石墙面图像——
灰色石砖纹理、青苔斑点、细微裂缝，外观极其逼真。
屏幕表面有特殊哑光涂层，模拟石头的粗糙触感。

背面（朝向片场侧）：黑色LED面板拼接结构。
可见面板框架、电源线和信号线束、
散热格栅、检修口。
面板拼接处有约2mm的黑色缝隙。

状态展示（三连图横向）：
左（LD-01 石墙态）：正面视角，仿石图像完美显示，看不出是屏幕
中（LD-02 穿透瞬间）：屏幕中央出现像素撕裂效果——雪花噪点+竖条纹+画面撕裂，
  一个人形剪影正在穿过屏幕中央
右（LD-03 片场暴露）：从背面视角看，LED面板中央有一块面板被推开/分离，
  透过缝隙可见冷宫侧的烛光

标注：
- "正面：仿石图像（可穿透柔性面板）"
- "背面：LED面板拼接结构"
- "穿透后不可逆（单向）"
- "安全：柔性面板，无碎玻璃或锐利边缘"

棚拍工作灯光，正面和背面分开照明。中性灰背景。
```

## 负向约束（全部道具通用）
- 不要CG渲染感过强，保持实拍道具质感
- 不要出现真实品牌Logo
- 比例尺/标注文字使用中文
- 状态图用小图并排展示，不要单张大图
