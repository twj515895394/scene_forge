# 配角图片提示词：导演 v1

## 角色设定图 Prompt

```
电影级角色设定图，经典棚拍武侠电影感，真人电影质感。

中国中年男性，40-50岁，微胖体型，圆脸双下巴初现。
戴深色棒球帽（鸭舌帽，纯色无Logo），帽檐压低。
穿深色导演马甲（多口袋，卡其色或军绿色），内搭深色Polo衫。
右手持黑色对讲机，天线竖起。
坐在帆布折叠导演椅上，面前是监视器屏幕（可见画面中有古装人物模糊影像）。
左手伸向监视器方向，仿佛正要按下回放键。

表情：严肃而专注，眉头微锁，嘴角平直——越认真越有喜剧感。

半身中景，正面略偏3/4，监视器屏幕在画面一侧可见。
片场工作灯光照明：冷白LED为主，混有暖色柔光。
```

## 表情变体

```
同一角色，三种表情并排：
1. 严肃专注（主表情）：盯监视器，眉头微锁，对讲机靠近嘴边
2. 惊喜认可：眉毛扬起，嘴角上翘，竖起大拇指——"这条情绪到了！"
3. 困惑转头：歪头，眉头皱起，对讲机放下——"她穿到B组了？"
```

## 负向约束
- 不要与任何真实导演撞脸
- 鸭舌帽/M马甲不要任何品牌Logo
- 不要卡通风格
