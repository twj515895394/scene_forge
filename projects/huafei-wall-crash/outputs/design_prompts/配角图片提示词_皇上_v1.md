# 配角图片提示词：皇上 v1

## 角色设定图 Prompt

```
电影级角色设定图，经典棚拍武侠电影感，真人电影质感。

中国中年男性，45-55岁，微胖体型，圆脸，皮肤偏白。
关键面部特征：八字胡+山羊胡（经典四郎胡子造型，参考陈建斌版雍正，
上唇两撇细长八字胡向两侧延伸，下巴一撮短山羊胡，胡须黑色中夹杂几根灰白）。
身穿明黄色龙袍（清代皇帝类型化简化版——宽袖、圆领、前胸有方形补子，
但补子纹样为抽象云纹而非具体五爪龙纹，避免复刻任何剧集的具体绣样）。
头戴红绒顶夏朝冠（简化版）。

坐姿，坐在深色雕花龙椅上，右手持iPad（现代平板电脑，黑色保护壳）。
表情：正在憋笑——嘴唇紧抿成一条线，腮帮微微鼓起，眼角已有笑纹，
眼睛盯着iPad屏幕，屏幕反射的微光映在他脸上。

半身中景，正面视角，龙椅靠背和帷幔在背景中。
暖黄宫灯光照明，柔光包裹人物。
```

## 表情序列（4连拍）

```
同一角色，四个连续表情横排（用于B06彩蛋）：
1. 憋笑憋不住：嘴唇紧抿成线，嘴角一侧微微上翘，腮帮鼓起，眼角肌肉抽动
2. 噗嗤破功：嘴微张，露出上排牙齿，眼睛眯起，鼻孔微张，肩膀微微抖动
3. 破防大笑：张嘴露齿大笑，眼睛眯成两条缝，头向后仰，iPad差点脱手
4. 笑得滑下龙椅：身体歪斜，一只手抓住龙椅扶手但已滑落一半，龙袍皱成一团
```

## 负向约束
- 龙袍补子使用抽象云纹，不要五爪龙纹
- 不要复刻任何具体剧集的龙袍款式细节
- 胡子的形状参考四郎但不要完全复制任何剧照
- 不要卡通风格
