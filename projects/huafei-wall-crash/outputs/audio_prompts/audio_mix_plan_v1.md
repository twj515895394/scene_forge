# 混音计划 v1

## 音轨优先级

| 优先级 | 音轨类型 | 说明 |
|--------|----------|------|
| 1(最高) | 台词/独白 | 干声，清晰度优先，不被任何声音压过 |
| 2 | 关键道具音效 | 橡皮鸭/boing/纸板/像素撕裂——这是笑点的听觉本体 |
| 3 | BGM | 服务情绪，关键笑点前主动"让位" |
| 4 | 环境音 | 铺底，不抢戏 |
| 5(最低) | 弹幕音效/细微环境 | 极轻，几乎感觉不到但存在 |

## 段间过渡混音

| 桥接 | 技术处理 |
|------|----------|
| Seg1→Seg2 | BGM轨在A04尾保持ff→B01撞击瞬间BGM轨mute(0.05s内拉到底)。橡皮鸭音效+柱子震颤独占0.5s，然后环境音轨从-∞渐入至-24dB |
| Seg2→Seg3 | Seg2尾B04环境音轨保持-30dB→C01拍灰音效在-12dB打破。无BGM轨过渡。环境音轨连续 |
| Seg3→Seg4 | C04凝固3秒所有轨-∞→D01撞桌瞬间BGM轨从-∞跳到-6dB+桌碎音效-3dB。静→爆无过渡 |
| Seg4→Seg5 | BGM轨连续(D06奥运BGM)→不切。E01俯冲时蹦床boing音效叠加在BGM之上(-3dB vs BGM -6dB) |
| Seg5→Seg6 | E05小鸟音效-6dB→0.5s所有轨-∞→F01衣料摩擦从-∞渐入至-18dB。BGM轨已归零。环境音轨从-∞渐入至-30dB(极轻) |
| Seg6→Seg7 | F03衣物摩擦-12dB→BGM轨从-∞渐入(ppp,-36dB)持续2s→G01画面淡入时BGM到达-24dB→继续渐强。环境音轨从冷宫(-30dB)交叉淡入到户外(-24dB) |
| Seg7→Seg8 | BGM轨连续不切，从-6dB(G05)推向0dB(H01)再推向+3dB(H02极限)→H03穿透瞬间BGM轨mute(0.1s内拉到底)。环境音轨交叉淡入(冷宫户外→片场内部) |
| Seg8→Seg9 | H05掌声轨从-3dB渐弱(reverb tail延长2s,用混响插件decay time调至3s)→掌声尾音与I01钢琴音头重叠0.3s。环境音轨交叉淡入(片场→休息区) |
| Seg9切黑→彩蛋 | I03切黑所有轨-∞持0.5s→I04 pizzicato从-∞跳到-6dB。环境音轨切换(休息区→养心殿安静) |

## 静默管理

| 静默点 | 时长 | 类型 | 处理 |
|--------|------|------|------|
| Seg2 BGM截断后 | ~3s | 喜剧静默 | 仅环境音-30dB，突出士兰"啊？" |
| Seg3 C04凝固对视 | ~3s | 社死静默 | 所有轨-∞，完全无声，直到吸面"咻—" |
| Seg6全段 | ~10s | 情感静默 | BGM轨-∞全程。环境音-30dB以下。独白干声独占 |
| Seg8穿透后 | ~2s | 爆炸静默 | BGM骤停后所有轨-∞持1s→环境音从-∞渐入 |
| Seg9切黑后 | ~0.5s | 节奏静默 | 所有轨-∞，为pizzicato的突然切入制造最大反差 |
