# 配乐生成提示词 v1

## 主题一：悲怆弦乐（Seg1）

```
电影级管弦乐，中国古装悲剧风格。
大提琴+小提琴为主，中速渐强(tempo 60→90bpm)。
开头低沉压抑(大提琴独奏，pp)→中段情绪积累(小提琴加入，mf)→
高潮全力释放(全弦乐+定音鼓，fff)→在高潮顶点被非音乐音效突然"切断"。
调性：D小调。情绪：从压抑到绝望爆发。
参考情感：甄嬛传式宫廷悲剧配乐。
时长：约10秒(实际使用00:00-00:10)，但需在10秒处被"拦腰截断"。
```

## 主题二：快节奏喜剧（Seg4）

```
电影级管弦乐喜剧变奏，轻快调皮风格。
木管+小提琴拨弦+定音鼓轻击，快速 tempo 140bpm。
每1.5-2秒一个小节，配合四道具失败节奏。
每个小节结尾留0.2s呼吸口(给道具音效让位)。
调性：G大调。情绪：忙碌、混乱、停不下来。
参考情感：卡通追逐场景配乐，但使用真实管弦乐音色(非合成器)。
时长：约6秒(实际使用00:30-00:36)，末尾无缝过渡到奥运BGM。
过渡：最后0.5s铜管进入，预示奥运主题。
```

## 主题三：奥运BGM变奏（Seg4尾→Seg5）

```
管弦乐奥运主题曲风格变奏，铜管+定音鼓+弦乐齐奏。
开头"登登登登～"为经典奥运开场动机的喜剧变奏版。
进入Seg5后每次boing弹跳加入pitch bend下滑(滑稽变调)。
弹跳1-3:正常音高+响亮。弹跳4-6:pitch微下滑+渐弱。弹跳7-8:pitch明显下滑+沉闷。
最后弹跳结束→渐弱→无声。
调性：C大调→弹跳中走调。情绪：从庄重仪式→滑稽失控→无力。
时长：约13秒(实际使用00:36-00:49)，末尾渐弱至无声。
```

## 主题四：史诗交响（Seg7→Seg8）

```
电影级史诗管弦乐，体育电影/英雄出征风格。
弦乐泛音极弱起(ppp)→铜管进入(mf)→定音鼓+弦乐全奏(ff)→推到极限(ffff)。
在ffff顶点突然静音(不是渐弱，是0.1秒内归零，类似电源被拔掉)。
静音后2秒完全无声→低音量环境音进入。
调性：降E大调。情绪：从希望→仪式→绝对信念→瞬间崩塌。
时长：约20秒(实际使用01:00-01:16)，01:16处突然静音。
静音前2秒(BGM峰值):所有声部齐奏，定音鼓滚奏，铜管高音长音保持。
```

## 主题五：温暖钢琴+喜剧pizzicato（Seg9）

```
A段(士兰收尾):温暖治愈钢琴独奏+轻柔吉他，tempo 70bpm，松弛。
G大调。情绪:释然、平静、"一切都过去了"。
I03"切黑"后钢琴渐弱至ppp→完全无声(1s)。

B段(皇上彩蛋):喜剧拨弦乐pizzicato strings急入，调皮+轻快+突然。
tempo 120bpm。配合憋笑→破功→大笑→滑下的四阶段递进。
每个阶段配乐强度递增(憋笑:单拨弦→破功:弦乐齐拨→大笑:全乐队→滑下:滑音下滑)。
定格时所有声音骤停(0.1s)→字幕滚屏:回到A段温暖钢琴+弦乐，形成闭环。
时长：约10秒(实际使用01:20-01:30)。
```
