# 拟音生成提示词 v1

## 拟音风格总控

- **质感**：棚拍实录质感——道具音效应听起来像真实的片场道具发出的声音，非CG合成
- **喜剧化程度**：允许轻微夸张（橡皮鸭/boing/小鸟），但不使用纯卡通合成音效
- **录音距离**：近距→中距交替。撞击/弹跳近距（加强冲击感），环境音中距

## 核心道具音效清单

### 泡沫柱撞击（Seg2 B01）
```
高密度泡沫被人体撞击的声音。
主体:沉闷的"砰"(低频,泡沫吸收冲击)→立即跟随"叽——"(高频,橡皮鸭玩具被捏爆的滑稽声)。
两个声音几乎同时但"叽"稍晚0.05s——让人先以为是真的撞击，然后……
橡皮鸭声:橡胶材质挤压+空气快速排出的短促尖叫，持续约0.3s。
撞击后柱子弹性震颤余音:低频嗡声,约1s后消失。
整体响度:撞击"砰"中等(mf)，橡皮鸭"叽"响(f，抢戏)。
```

### 纸板柱+敲击（Seg3 C02+C03）
```
指节敲击瓦楞纸板表面:"咚咚咚"，三下，每下间隔0.3s。
音色:空洞+清脆，中低频为主，有纸板特有的轻微回响。
不像敲木头(沉闷)，不像敲金属(尖锐)，就是"纸板声"。
柱子倒下:"啪嗒——"，干瘪纸板倒地的轻响，无回弹，无余震。
纸板落地后轻微弹跳一下再静止(二次微响)。
```

### 自动感应门（Seg4 D02）
```
超市/商场自动感应玻璃门的标准音效。
"叮咚——"清脆双音(高音"叮"+低音"咚")，持续0.5s。
门电机滑开声:"嗡——"低频持续1s，然后停止。
不需要"欢迎光临"的人声播报——用纯音效留白更好笑。
```

### 充气香炉瘪（Seg4 D03）
```
充气塑料玩具被快速挤压排空的声音。
"噗——"，中频，持续0.3s。
类似气球放气但更闷(因为香炉外壳有布料/塑料涂层)。
空气排空后塑料塌陷的细微"沙沙"声(塑料自身皱褶摩擦)。
```

### 绿幕布缠绕（Seg4 D04+D05）
```
大幅棉质布料从高处脱落:"哗啦——"，中低频，持续0.5s。
布料缠绕人体滚动:连续"沙沙沙"摩擦声(棉布+地面+人体)，持续1.5s。
布料撕裂:"嘶——"，中高频，短促0.2s(士兰挣脱时)。
```

### 蹦床弹跳×8（Seg5）
```
真实蹦床弹簧声:金属弹簧拉伸+回弹的"boing"，每个0.2s。
8次弹跳，每次音高递减:
弹1-2: C5(响亮,高频充足)
弹3-4: B4(稍降)
弹5-6: A4(渐弱)
弹7: G4(沉闷)
弹8: E4(最沉闷,几乎只有低频)
非卡通合成音效——使用真实蹦床录音+轻微音高调整。
弹跳间穿插士兰喊叫(非拟音，人声)。
```

### 小鸟星星（Seg5 E05）
```
卡通小鸟叫声:"啾啾啾～"，3声，短促清脆。
经典卡通晕倒音效风格，但不过度合成。
金色星星闪烁无声音(纯视觉)，但小鸟叫声作为听觉锚点。
```

### 鸽子飞过（Seg6 F02）
```
鸽子翅膀扑棱:"噗噜噜——"，中低频，持续0.3s。
鸽子叫:"咕咕～"，清脆中音，一声。
这两个声音是Seg6全静默中唯一的听觉事件——所以必须干净、清晰、不拖泥带水。
```

### LED穿透（Seg8 H03）
```
数字设备故障音效组合:
- 像素撕裂:高频"滋滋——"持续0.3s
- 扫描线蜂鸣:中频持续"哔——"0.5s
- 雪花噪点:白噪短爆"沙"0.1s
三种声音几乎同时→交叠→0.5s后全部归零(与BGM同步骤停)。
非恐怖/非身体伤害音效——它是"屏幕坏了"，不是"人受伤了"。
```

### 土兰昏倒（Seg8 H05）
```
身体前倾倒地:衣物摩擦"沙"+身体撞击地板"咚"(闷,中低频)。
与Seg2弹回坐地不同——这次是真的昏倒，身体完全无力。
"咚"应比"砰"更沉闷、更重，持续0.3s。
```

## 环境音清单

| 环境音 | 使用段 | 描述 |
|--------|--------|------|
| 冷宫烛火 | Seg1-6 | 烛火噼啪微响(间歇,低频)，高窗微风"呼——" |
| 片场低频嗡 | Seg6(F01-F02背景) | 空调/设备低频持续嗡声，极轻(几乎不可闻) |
| 片场远处人声 | Seg6(F02背景) | 模糊的对话碎片，像来自另一个房间 |
| 户外微风+鸟鸣 | Seg7 | 室外自然风"呼——"+远处鸟鸣 |
| 片场设备 | Seg8(H04起) | 设备低频嗡+对讲机杂音+空调+偶尔的脚步声 |
| 片场收工 | Seg9(I01-I03背景) | 拆道具声(泡沫/金属/布料)、卷线声、模糊人声 |
| 养心殿安静 | Seg9(I04-I05) | 极静的室内——无风无设备，只有皇上自己的声音 |
