# 导演分镜故事板清单

*   **项目名称**：`million-pound-biao`
*   **版本**：v1.0
*   **分镜阶段**：`scene-storyboard-director`
*   **镜头总数**：18个镜头 (9个 10秒分段，均匀技术分段)
*   **导演风格**：`exaggerated_comedy` (夸张动作喜剧)
*   **空间划分连续性 (Blocking Map)**：
    *   **Zone B (左侧)**：收银柜台，吴总与阿刚的大本营。
    *   **Zone A (中央)**：正大门，罗马挂钟与送客/走火区域。
    *   **Zone C (右侧)**：靠窗圆桌，彪哥与马大帅（就餐与亮卡区）。
*   **道具状态规则**：
    *   **至尊黑卡**：钱包内置 -> 抽出金芒一角 -> 圣光展开 -> 塞回包包。
    *   **包裹猎枪**：红绿花布裹紧斜跨 -> 绳散坠落裸露枪管 -> 触地走火青烟 -> 裹回花布。

### 🎬 Segment 1 (00:00 - 00:10) | Beat B01: 窗外犹豫
#### 🎥 Shot 01 (00:00 - 00:05)
*   **所属 Beat / Unit**：B01 / Unit 1.1 (setup)
*   **景别 / 机位 / 角度**：大特写 (Extreme Close-up) / 固定机位 / 平视 (Eye Level)
*   **构图 / 镜头运动**：三分法构图 (窗口冰霜花纹偏左)，镜头完全静止。
