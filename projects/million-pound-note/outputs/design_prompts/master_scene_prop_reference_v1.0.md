# 全场景资产总参考图 Prompt 包 v1.0

本包包含两幅全场景资产总参考图（Master Scene-Prop Reference Board）的 3D 渲染 Prompt，用以统一多角色、多道具在同一空间下的灯光、色调、站位和构图关系。

---

## 📌 Master Prompt 1：对峙与窥视（进食与催债铺垫阶段）

### 描述（中文主导版）
> **场景与构图**：
> 3D全景场景概念设计图，皮克斯电影级动画风格。
> 画面采用略带俯视的对角线构图。
> 
> **空间细节与角色站位**：
> *   **右下角前景（Zone C）**：一个面容平静慈悲、身穿破旧补丁土灰色僧袍的年轻中国僧人（唐朝和尚），坐在铺着白色餐桌布的木圆桌旁，正专注地端着青陶大碗，用竹筷子大口吃素面。桌旁放着他的破竹笈。
> *   **左侧中景（Zone B）**：高耸的胡桃木结账柜台。柜台后站着矮胖驼背、身穿呢绒马甲挂着大颗红宝石念珠的老掌柜（金池长老），以及一个身穿黑色燕尾马甲手搭毛巾的瘦长小二。两人躲在柜台木柱子后，正用嫌弃、市侩和睥睨的冷漠眼神，偷偷窥视右侧卡座里吃面的唐朝和尚。
> *   **背景深处（Zone A）**：双开的木质拱形大门，大门上方悬挂着大黄铜壁挂钟。
> 
> **灯光与色调**：
> 低饱和度、冷暖对比色调。背景为古典英伦深色胡桃木墙板。柜台上方有温暖的壁灯照耀，而唐朝和尚所在的右侧角落则处于柔和的阴影中，窗外透进一缕蓝灰色的雾光。整个画面呈现英伦茶室的优雅与东方落魄和尚吃面的荒诞反差。
> 
> **风格锚词**：
> 3D master scene reference, Pixar cinematic animation style, funny contrast, Victorian English restaurant interior, character blocking map, warm gas lamp lighting vs blue window light, rich fabric textures, Octane rendering. --ar 16:9

## 📌 Master Prompt 2：圣光降临（通关文牒反转高潮阶段）

### 描述（中文主导版）
> **场景与构图**：
> 3D中景戏剧反转渲染图，皮克斯电影级动画风格。
> 画面采用略带低机位的仰角拍摄，聚焦于白色圆餐桌。
> 
> **道具状态与角色反应**：
> *   **视觉焦点（Zone C 桌面上）**：餐桌的白色棉布台布正中央，平整地铺开着一张由纯金丝线织就的明黄色大唐黄金通关文牒，上面有遒劲的黑字和一枚巨大的朱红玺印。文牒正爆发出夺目、温暖的**金色体积光晕 (Volumetric Golden Glow)**，体积光呈扇形向斜上方辐射，空气中漂浮着无数金色微光粒子。
> *   **右侧角色（唐朝和尚）**：唐朝和尚端坐在椅子上，双手合十，神态一如既往地安详平静，金色的圣光将他清秀的脸庞镀上一层温暖的淡金色轮廓光。
> *   **左侧角色（金池长老与店小二）**：金池长老和小二扑在桌旁。老掌柜满脸冷汗，单片金丝眼镜被金光气流震得飞在半空，他正双手打颤、极度惊愕地拿着放大镜盯着金牒；小二由于极度惊骇，下巴滑稽地脱臼拉长（Jaw Drop stretch），眼睛凸出眼眶。
> 
> **灯光与色彩**：
> 金色圣光是全场唯一的核心强光源，照亮了惨白的餐桌布，并将贪婪老僧和小二的惊恐脸庞映照成金黄色。背景的胡桃木墙壁和悬挂的挂钟在圣光的强烈对比下被压成深色的暗影剪影。
> 
> **风格锚词**：
> 3D master scene reveal, epic reverse shot, Pixar cinematic animation climax, magical glowing scroll on white tablecloth, warm volumetric rays lighting up shocked cartoon characters, stunning visual payoff, high fidelity render. --ar 16:9
