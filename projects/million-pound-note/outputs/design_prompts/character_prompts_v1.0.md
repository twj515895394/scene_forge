## 1. 角色 1：唐朝和尚 (Tang Dynasty Monk)

### 📌 Prompt（中文主导版）
> **描述**：
> 3D角色设计设定板，皮克斯电影级动画风格。主角是落魄的东方年轻僧人唐朝和尚，光头，清秀、平静慈悲的面部特征，细长清澈的双眼，面带淡淡的温和微笑。身穿一件发白、打满粗布补丁、带有磨损毛羽纤维的土灰色粗布僧袍。背着一个简陋的竹编笈，笈框上挂着一个洗脸铜盆和一卷发黄宣纸。
> 
> **板式结构**：

> 1. 多视角全身三维参考模型图（正面、3/4侧面、背面）。
> 2. 表情区：展示6个不同的剧情表情（慈悲安详的微笑、专注大口吸面、肚子饿时的微蹙眉头、平静合十的淡然、从袖口摸索东西的憨厚、被金光映照脸庞的圣洁神态）。
> 3. 服装配件细节区：特写补丁僧袍材质、简陋的竹筷、破旧的竹笈。
> 
> **灯光与风格锚词**：
> 3D Pixar character sheet, feature animation style, clay model render texture, detailed fabric normal map, cinematic soft studio lighting, high-resolution rendering, Unreal Engine 5 render style. --ar 16:9
