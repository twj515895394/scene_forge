# 角色视觉设计锁定卡 v1.0

本项目为《百万英镑》经典餐厅结账片段的西游跨界改编。所有角色设计均继承统一的 3D 动画电影级（皮克斯风格）设计语言，形状高度概括，比例夸张，但表面材质追求电影级的真实感与精细度。

---

## 1. 统一角色设计原则与特效表现

*   **设计基调**：Pixar 风格三维动画角色设计。头部与五官比例适当放大以增强表情张力，眼睛具有清晰的眉骨和灵动的虹膜结构，手部结构简化为略微饱满的四指，动作具备丰富的物理拉伸与回弹表现。
*   **出场名字显示规则 (On-Screen Name Tag)**：
    *   根据用户指示，每个主要角色首次出场时，画面中央下方需以浮雕质感或古风手写毛笔字样式显示角色姓名（例如：“唐朝和尚”、“金池长老”、“店小二”），并配以短暂的滑音或木鱼声特效。
*   **表现力动作策略 (Expressive Animation)**：
    *   允许面部和身体产生强烈的卡通物理形变（如极度拉长的下巴、脱出眼眶的眼球、吸气时膨胀的胸腔）。
    *   动作速度极快时允许使用 Smear Frame（运动拖影残影），角色受惊时可产生短暂的贴墙或纸片化回弹，但必须在搞笑结束后立即恢复角色原始的视觉完整性。

## 2. 唐朝和尚 (Tang Dynasty Monk)

> **角色定位**：扮猪吃老虎（隐秘大佬）。表面是落魄寒酸的流浪僧人，实则是手握大唐皇帝亲笔及国家财富信用背书的御弟圣僧。

### 2.1 形状与比例
*   **头身比例**：标准 6 头身。身形偏向清瘦、挺拔，站姿稳重，行走缓慢而神态安详。
*   **形状语言**：以直线、稳固的长方形和温和的圆角矩形为主。下巴线条柔和，眉骨平整，具有东方古典僧侣的清秀与出尘感。
*   **脸部特征**：参考经典中式圣僧脸谱。眉毛为秀气的剑眉，双眼清澈微合，嘴角常挂着一抹慈悲平静的微笑。即使在吃素面时，面部神情依然干净、慈祥，与狼吞虎咽的动作形成反差喜剧。

### 2.2 材质与服饰
*   **僧袍材质**：穿着一件发白、打满粗麻布丁的土灰色僧袍。僧袍表面有清晰可见的麻线交织纹理，袖口和下摆有轻微的磨损、起毛边缘，衣角沾有少许路途尘土。
*   **配件**：
    *   **竹笈**：背上背着一个用粗竹条编织的简陋笈框，里面装有一只黄铜洗脸盆、几卷破旧的宣纸经书和一壶挂在笈边的葫芦水瓶。
    *   **竹筷**：随身携带的一双木质旧竹筷，筷子表面因长期使用而呈现深褐色的木纹磨损光泽。
*   **色彩配方**：低饱和度的土灰色、枯叶褐色、沙土黄，整体色调暗淡，在昏暗的茶室卡座中显得尤为寒酸。

### 2.3 表情与姿态板 (6-9 个表情)
1.  **平静慈悲 (默认)**：嘴角微扬，双眼微合，神态如古佛般安详。
2.  **饥肠辘辘**：肚子发出“咕噜”特效声时，眉头微皱，嘴唇微张，视线向下盯着餐桌。
3.  **吃面专注**：双眼放光，嘴塞得鼓鼓的，双颊有明显的卡通充气感，大口吸面。
4.  **淡然一笑**：面对小二递来的账单和拍桌子逼债，没有任何愤怒，微微合十点头，神情自若。
5.  **伸手摸索**：从破僧袍袖口中掏出信封时，微微眯眼，带着一丝憨厚和从容。
6.  **展示文牒**：神态平静而庄重，双手将黄金文牒展开在台布上，金光映照其脸庞。
7.  **起步道别**：双手合十，微微躬身，慈悲点头，踏出茶室时神态自若。

### 2.4 物理与安全边界
*   允许在吃面时有面汤甩起的小水滴特效，嘴边粘有少许葱花和面屑（Minor dirt）。
*   绝对禁止任何因饥饿导致的写实虚脱、生病或流血状态。

## 3. 金池长老 (Elder Jinchi)

> **角色定位**：势利眼高级餐馆老板。崇拜财富与权势，为人虚荣、刻薄，具有极度傲慢与极度谄媚的戏剧反差。

### 3.1 形状与比例
*   **头身比例**：约 5 头身。身形肥胖、有些驼背，肚子微微凸出，显得重心下移。
*   **形状语言**：以大圆形、松软的椭圆形和不规则波浪线为主。脸颊肥厚下垂，眉毛倒竖成八字。
*   **脸部特征**：白色八字胡，满脸市侩的褶皱，在笑的时候眼睛眯成一条缝，怒的时候眼球凸出。戴着一副单片金丝放大镜，放大镜用细细的金丝链条挂在耳朵上。

### 3.2 材质与服饰
*   **中西合璧服饰**：
    *   **呢绒西装马甲**：穿着英式绅士的深红呢绒马甲西装，马甲纽扣由纯金打造，散发金色反光，口袋里插有一块绣着牡丹花的丝绸手帕。
    *   **红宝石佛珠**：马甲外面斜挎着一串极其夸张、奢华的大红色佛珠。每颗佛珠有乒乓球大小，由质地纯净的红宝石制成，表面如镜面般水润，折射着茶室里壁灯的红光。
*   **色彩配方**：高饱和度的深红色、亮金色、墨黑色，整体色彩华贵而俗气。

### 3.3 表情与姿态板
1.  **傲慢睥睨 (默认)**：单片放大镜微微反光，下巴抬高，斜眼看着角落里的唐朝和尚。
2.  **算计嘀咕**：嘴角向下扯，眼睛眯起，用手指拨弄算盘，露出嫌弃和市侩的表情。
3.  **愤怒施压**：拍桌子催债时，眉毛倒竖，脸憋得通红（Red pressure aura），嘴张得极大，露出牙齿。
4.  **怀疑质问**：看着破信封里的通关文牒时，单眼放大镜凑近，眼睛瞪大，满脸怀疑。
5.  **瞳孔巨震**：确认是黄金文牒真迹后，瞳孔瞬间缩小，眼眶放大，嘴巴张成 O 型。
6.  **魂飞魄散**：双腿打颤，满脸流虚汗（卡通大汗滴），单片眼镜吓得飞出。
7.  **极度谄媚**：满脸堆笑（褶皱挤在一起），腰弯成 90 度，双手作揖，嘴巴咧到耳根。

### 3.4 物理与安全边界
*   拍桌子时允许震起桌面的微弱白色尘雾团（Dust puffs），且允许他在震惊中单片眼镜被震飞。
*   禁止任何写实的跌倒摔伤，长跪不起或下跪时必须是滑稽、软性的橡皮泥般动作回弹。

## 4. 店小二 (Waiter - 金池弟子)

> **角色定位**：市侩的跑腿侍应生。狗仗人势，执行催债的前线马前卒，承担最夸张的喜剧动作反应。

### 4.1 形状与比例
*   **头身比例**：约 5.5 头身。身材极度瘦长，手臂与双腿如同长面条般细长，方便做出大范围位移动势。
*   **形状语言**：以细长线条、尖锐三角形和不稳定的折线为主。肩膀耸起，动作慌乱局促。
*   **脸部特征**：眼距较宽，大门牙微微凸出，戴着一顶西式圆顶侍应帽，五官具有极强的弹性形变空间。

### 4.2 材质与服饰
*   **服饰细节**：身穿紧身黑色双排扣马甲，但领口和袖口缝有中式僧侣服饰的浅灰色滚边。手里搭着一条白色的搭线毛巾（毛巾材质粗糙吸水，在逼债时被他神经质地揉捏）。
*   **色彩配方**：黑白分明的侍应生制服色，点缀灰色滚边，融入茶室的暗色调背景。

### 4.3 表情与姿态板
1.  **狗仗人势 (默认)**：斜眼瞪人，双手叉腰，抹布甩在肩膀上，神气活现。
2.  **催账逼债**：身体前倾压向唐朝和尚，将账单死死拍在桌上，眉毛挑起，满脸得意。
3.  **下巴脱臼 (Shock)**：看到黄金通关文牒的玉玺红印瞬间，下巴夸张地拉长到胸口（Jaw drop stretch），嘴巴大张，眼睛凸出眼眶。
4.  **惊慌碎步 (Panic Scramble)**：捧着黄金文牒像捧着炸弹一样，双脚原地打滑转圈（卡通小碎步），然后飞速滑向金池长老。
5.  **恭敬谄媚**：双手托着通关文牒，低头哈腰，面部挤出夸张的月牙状笑容。

### 4.4 物理与安全边界
*   允许大范围的物理形变（下巴脱臼拉长、贴墙扁平、碎步拖影）。
*   严禁任何流血、撕裂或身体恐怖（Body horror）的不适形变，形变后必须有明显的橡皮回弹声效（Boing）并恢复原样。
