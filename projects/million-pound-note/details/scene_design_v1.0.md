# 场景视觉设计锁定卡 v1.0

本项目场景继承原片经典的 20 世纪初高档英式茶室（物理场地保持完全一致）。设计旨在建立一个极具质感、优雅深沉的西方英伦空间，与落魄的东方僧人及闪耀着大唐圣光的黄金通关文牒形成最强烈的视觉文化反差。

---

## 1. 空间布局与物理连续性约束 (Blocking Map)

为了保证镜头切换时的空间感（物理连续性），场景空间划分为三个核心区域（Zone A, B, C），沿一条明确的左右物理轴线分布：

```
[左侧 Zone B: 柜台与账台] <===========> [中央 Zone A: 大门入口] <===========> [右侧 Zone C: 靠窗餐桌卡座]
    (金池长老常驻地)                         (对称木门, 挂钟下方)                   (唐朝和尚吃面桌)
```

### 1.1 区域明细
*   **Zone A (大门与挂钟区)**：
    *   位于画面中后部。一扇精美的双开拱形深色橡木木门，门上方装有弧形彩绘玻璃窗。
    *   木门正上方悬挂着本片的核心视听道具——**复古黄铜壁挂大钟**。挂钟表盘为象牙白，罗马数字，钟摆在重力作用下左右摇晃，发出沉重而沉闷的“滴答、滴答”声。
*   **Zone B (账台柜台区)**：
    *   位于画面左侧。高耸的深色胡桃木账台，柜台表面打磨得光滑如镜，泛着微微的冷光。
    *   账台上摆放着一本黑皮大账本、一尊墨绿色的玉石算盘，以及一盏绿铜雕花台灯。
*   **Zone C (靠窗卡座区)**：
    *   位于画面右侧。一张铺着精细白色织物台布的胡桃木圆桌，配有一把深色皮革软包扶手椅（唐朝和尚的座位）。
    *   圆桌一侧有一扇高大的维多利亚式推拉格子窗，窗外是朦胧的伦敦街景雾气。

## 2. 材质与视觉细节

*   **胡桃木墙板**：墙壁镶嵌着温润的深胡桃木护墙板，墙板带有细腻的欧式线脚，表面涂有微哑光的清漆，能隐约反射出壁灯的橘黄色光晕。
*   **白色台布**：白色餐桌布材质为厚实的粗棉布，经纬线清晰，桌角挂着手织蕾丝边，与唐朝和尚质朴粗糙的形象以及黄金文牒的闪亮色彩形成极好的材质对比度。
*   **金属与玻璃**：壁灯的古铜托架、黄钟的黄铜外壳、餐具的银质反光、格子窗的微瑕老玻璃，均具备极高的材质物理反射效果（PBR）。

## 3. 灯光与色彩剧本 (Color Script)

场景灯光服务于反差戏剧性的递进：

### 3.1 阶段一：昏暗压抑（吃面与催账期）
*   **光源配置**：关闭全局大灯，仅开启墙壁上的橘黄色古典壁灯和柜台台灯，窗外透入灰蓝色的英国冷雾光。
*   **光影效果**：在角落的 Zone C（唐朝和尚处）形成明显的投影区。唐朝和尚融入土黄与棕褐色的暗影中，显得微弱、局促。而金池长老和小二从 Zone B 的暖橘光中向 Zone C 逼近，拉出长长的压迫性投影。

### 3.2 阶段二：圣光爆开（展开通关文牒）
*   **光源配置**：文牒展开的瞬间，文牒表面的纯金线在空气中触发类似迪士尼/皮克斯特效的**暖金色物理体体积光晕 (Volumetric Golden Glow)**。
*   **光影效果**：这股强烈的金色圣光呈扇形爆开，瞬间照亮 Zone C 的白色台布，并向上投射，强烈地照亮金池长老与小二错愕的脸庞（在其眼珠和脸上折射出耀眼的金色高光）。此时，背景的深色胡桃木墙板和深处的大挂钟在金光对比下被压暗成剪影，视听焦点实现降维式聚拢。

## 4. 空间一致性防错规则

1.  **挂钟位置绝对锁定**：无论使用何种机位，大挂钟必须始终处于 Zone A 大门上方。
2.  **左右关系绝对锁定**：从摄像机正对大门视角看，结账柜台（金池处）必须在左侧，唐朝和尚的白台布餐桌必须在右侧。严禁在后续分镜中产生方向倒置。
3.  **窗户朝向绝对锁定**：Zone C 的推拉格子窗必须位于唐朝和尚的右手侧（朝向右方）。
