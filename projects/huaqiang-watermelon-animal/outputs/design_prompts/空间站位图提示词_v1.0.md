# 空间站位图片生成提示词 (空间站位图提示词_v1.0.md)

本文件包含用于在外部图像生成平台生成“空间站位图”参考图的最终 Prompt。其目标是为后续的分镜生成建立可供模型稳定继承的几何和角色相对位置控制图。

```text
【提示词类型】：空间调度控制板 (Spatial Blocking Reference Diagram)
【风格定位】：高品质 3D 动画电影线稿与色彩草图 (3d blueprint sketch with colors), 俯视视角 (semi-top-down perspective), isometric view
【主画面布局】：
一个半俯视角度、类似游戏关卡或 3D 空间三维站位参考图。画面用于指明水果摊对峙发生时各元素的几何分布。

【场景空间与轴线布置 (Spatial Axis & Landmarks)】：
- 中央视觉中轴：水果摊长木案板。
- 摊后操作区 (Seller's Zone)：位于木案板后侧，上方覆盖有红蓝雨棚。
- 摊前对峙区 (Buyer's Zone)：位于木案板前侧，是一条黄土碎石泥马路。
- 摩托车通道 (Motorcycle Path)：贯穿摊前路面，呈左至右水平走线。

【角色站位与相对位置标定 (Character Default Blocking)】：
- 角色A（买方 - 平头哥蜜獾）：位于左前方的马路空地旁，靠近他停放的粉色踏板摩托车。距离木案板约 1.5 米，身体面向案板（斜向右上）。
- 角色B（卖方 - 恶霸犬摊主）：站立在木案板的正后方中央，双手按在案板上，俯视着左前方的平头哥。
- 角色C（卖方跟班 - 土拨鼠小弟）：站在木案板的右后方（靠近电子秤的一侧），躬身贴着恶霸犬的侧后方。
- 核心道具（电子秤）：稳固摆放在木案板的右前侧边缘上。

【辅助引导线描述 (Visual Mapping Helpers)】：
- 箭头指向线：有一条从平头哥停摩托处指向案板前方的黄色虚线路径（代表行进轨迹）。
- 红色警戒圈 (Interaction Circle)：电子秤与案板中段连成一个红色的椭圆形物理交互敏感圈。
- 出口方向 (Exit Path)：摊前马路向左侧延伸出一条大箭头（指向退场方向）。

【负向约束 (Negative Prompt)】：
写实电影海报 (cinematic movie poster), 角色大头照 (portrait), 混乱的角度 (flat ground level view), 无空间关系 (no depth).
```
