## 1. 角色一：平头哥 (蜜獾) 角色说明书板 Prompt

```text
【提示词类型】：角色设定板 (Character Bible Sheet)
【风格定位】：高品质 3D 动画电影质感 (feature animation style), 皮克斯/小黄人角色设计基调 (3d character render)
【主画面布局】：
一张完整的 3D 角色设计参考板 (character design board, character model sheet)。白色背景，排版干净工整。

【角色主体描述】：
一个拟人化的 3D 卡通蜜獾 (anthropomorphic honey badger) 角色。
- 头身比：6.2 头身，强壮矮实。
- 头部特征：头部上方到背部是平整、亮眼的白色短毛发（形成经典的“白发平头”），面部为黑灰色短毛。
- 五官特征：眼睛比真人略微放大，上眼眶平直，眉骨用力下压，瞳孔小而锐利，眼神极度平静、冷酷、专注于对手。小嘴巴，左嘴角叼着一根干细的枯黄草签。
- 服装细节：身穿一件硬挺、合身、略显拉风的黑色复古机车皮夹克 (black leather biker jacket)，垫肩坚挺，拉链和金属扣有精致的拉丝金属折射质感。内搭贴身深灰色棉质背心。下身着磨损破旧的深蓝色牛仔裤，脚穿一双厚重粗糙的黑色高帮机车靴。

【板块分区展示 (Board Section Requirements)】：
1. 多视角参考区 (Multiple Views)：包含正立面 (front view)、3/4侧面 (three-quarter view)、背立面 (back view)。
2. 表情区 (Expressions)：
   - 冷静面瘫 (Neutral face, chewing straw)
   - 怀疑冷笑 (Suspicious gaze, single eyebrow raised)
   - 危险的死亡凝视 (Intense stare, low eyebrows)
3. 动作姿态区 (Action Poses)：
   - 跨坐在粉色踏板摩托车上熄火的姿态，头戴带萌萌猫耳装饰的粉色安全帽
   - 爪子敲击西瓜的姿态
   - 闪身挥爪、破空抓出白芒特效的动态瞬间
4. 细节展示区 (Detail Callouts)：机车皮夹克金属铆扣近景，手部锋利爪子的特写。
5. 比例对照组 (Scale Reference)：矮个子，与高大角色身形对比图。

【文字区说明】：
画面顶部及边角保留必要的文字信息区 (character name: "华强 - 平头哥" and labels for views).

【负向约束 (Negative Prompt)】：
流血 (blood), 写实伤口 (wounds), 现实真人脸 (realistic human face), 幼稚玩具感 (childish toy), 模糊虚化 (blurry), 杂乱背景 (cluttered background).
```

## 2. 角色二：恶霸犬 (摊主) 角色说明书板 Prompt

```text
【提示词类型】：角色设定板 (Character Bible Sheet)
【风格定位】：高品质 3D 动画电影质感 (feature animation style), 3d animation render, lighting language "warm afternoon light"
【主画面布局】：
一张完整的 3D 角色设计参考板 (character design board, character model sheet)。白色背景，明亮排版。

【角色主体描述】：
一个拟人化的 3D 卡通美国恶霸犬/斗牛犬 (anthropomorphic American Bully/Bulldog) 角色。
- 头身比：5.8 头身，横向身材极其肥胖粗壮，大腹便便。
- 头部特征：面部布满灰色皮肤褶皱，大红鼻子 (red bulbous nose)，大嘴巴垂在脸侧，呲牙时露出一排粗钝的犬齿。眼睛宽距，眼袋大，带有不耐烦的嚣张神色。
- 服装细节：穿着一件紧绷、略带泛黄和汗渍的白色纯棉吊带背心，左侧肩带有些许下垂。下身穿着松垮的军绿色老旧短裤，皮带上挂着一大串钥匙扣。脚踩一双深蓝色橡胶人字拖 (rubber flip-flops)。

【板块分区展示 (Board Section Requirements)】：
1. 多视角参考区 (Multiple Views)：包含宽壮的正立面 (front view)、3/4侧面 (three-quarter view)、背部肥大肚腩的后视角度。
2. 表情区 (Expressions)：
   - 市侩不耐烦 (Rough scowling face)
   - 气急败坏大吼 (Roaring in anger, red skin wrinkles)
   - 遭到大爆炸时的呆滞震惊 (Shocked face, tiny pupils)
   - 眩晕倒地 (Dizzy X_X eyes, bubbles at mouth, rotating golden stars floating above head)
3. 动作姿态区 (Action Poses)：
   - 躺在小竹椅上跷二郎腿剔牙的姿态
   - 恶狠狠抓起大西瓜刀指向前方的姿态
   - 被西瓜汁气浪击中、在空中陀螺旋转的动态反馈
4. 比例对照组 (Scale Reference)：宽壮高大的大肚身材，与矮小蜜獾的对比。

【文字区说明】：
画面顶部及边角保留必要的文字信息区 (character name: "摊主 - 恶霸犬" and labels for views).

【负向约束 (Negative Prompt)】：
出血 (blood), 写实伤口 (graphic wounds), 真人脸 (realistic face), 过于呆板 (stiff pose), 背景嘈杂 (busy background).
```
