# 主参考设定图片生成提示词 (master_scene_prop_reference_v1.0.md)

本文件包含用于在外部图像生成平台生成“全场景资产总参考图 (Master Board)”的最终 Prompt。它将场景、三角色身形对照、电子秤作弊状态整合进一张参考图中，便于在后续故事板与视频连段生成中锁定全宇宙的视觉一致性。

```text
【提示词类型】：全场景资产总参考图 (Master Scene-Prop Reference Board)
【风格定位】：高品质 3D 动画电影质感 (feature animation style), 3d scene design layout render, cinematic storytelling layout
【主画面布局】：
一张整合了主场景空间、角色比例和核心道具细节的 3D 动画电影级艺术总参考设定板 (master visual guide board)。白色背景，高级图表与画卷的混合排版。

【主画面三大核心板块 (Three Core Sections in Master Board)】：
1. 【中景主场景布局板 (Master Scene Layout)】：
   展示斜阳下的大排档街口水果摊全貌。左前方是停在泥路上的小踏板车与粉盔，中央案板桌后站着剔牙的恶霸犬，右侧缩着摇扇的土拨鼠，摊前平头哥站立。整体呈现 45 度温热夕阳与红蓝雨棚下的对峙张力。

2. 【角色比例对照组 (Character Scale Matrix)】：
   三个拟人角色站成一排以提供绝对比例对照。
   - 左侧为干瘦高条的土拨鼠 (6.8头身) ⬅️ 中间为矮健硬挺的平头哥蜜獾 (6.2头身，身穿垫肩黑色皮衣) ⬅️ 右侧为横向肥壮巨硕的美国恶霸犬 (5.8头身)。身形呈现滑稽的“瘦高-矮健-宽胖”戏剧反差。

3. 【核心道具物理矩阵 (Key Prop Mechanism)】：
   特写展示蓝色电子秤的三种状态细节。
   - 正常开机模式 ➡️ 秤底粘附着圆形特大吸铁石（露出合格标签贴纸）的作弊模式 ➡️ 秤盘飞翻落地状态。

【负向约束 (Negative Prompt)】：
真人照片风格 (realistic photos), 单个角色大头像 (portrait), 混乱的色调 (clashing colors), 没有透视关系 (flat projection).
```
