# 场景设计图片生成提示词 (scene_prompts_v1.0.md)

本文件包含用于在外部图像生成平台生成“街角水果摊”主场景参考图的最终 Prompt。

```text
【提示词类型】：场景设计设定图 (Scene Design Reference Board)
【风格定位】：高品质 3D 动画电影质感 (feature animation style), 温暖细腻的皮克斯电影级背景 (warm afternoon movie light, 3d scenery render)
【主画面布局】：
一个位于老旧繁忙中国南方街角的 3D 卡通水果摊 (corner fruit stand)。虚化的远景为大排档老街。

【场景细节描述】：
- 顶棚：一个红蓝条纹相间 (red-and-blue striped) 的旧塑料防雨遮阳棚挂在摊位上方，雨布有轻微的风化破损。
- 瓜架：摊位左侧摆有一台墨绿色生锈的三层铁质货架，架上整齐堆叠着墨绿条纹的饱满大西瓜。
- 案板桌：中央是一张宽大的老旧木质长桌，桌面呈现油腻木纹光泽，桌上散落着几个西瓜。案板右侧摆着一台蓝色外壳的双面数显电子秤。电子秤旁插着一把陈旧的水果砍刀。
- 地面：粗糙多尘的泥石路面，有少量树影斑驳。

【光影与氛围控制】：
- 灯光：焦灼热烈的夏日午后 45 度斜照强光 (warm late afternoon sun)。阳光在泥地上拉出长长的遮阳棚投影，边缘轮廓分明。
- 材质：粗糙木纹、反光金属拉丝电子秤秤盘、高蜡质西瓜皮、防雨布微小的纤维反光质感。

【负向约束 (Negative Prompt)】：
现实真人背景 (realistic photo), 纯白背景 (blank background), 室内超市 (supermarket interior), 现代高级商铺 (modern luxury shop), 模糊成一片 (flat render).
```
